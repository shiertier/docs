## 摘要

**1) 一句话总结**
Mistral AI 开放了其开发者平台 La Plateforme 的 Beta 测试，首批提供三个不同性价比的文本生成端点以及一个嵌入端点。

**2) 关键点**
*   **模型技术**：提供基于公开网络数据预训练及标注数据指令微调的模型，采用高效微调和直接偏好优化（DPO）技术进行对齐。
*   **Mistral-tiny 端点**：最具成本效益，基于可下载的 Mistral 7B Instruct v0.2 模型，仅支持英语，MT-Bench 得分为 7.6。
*   **Mistral-small 端点**：基于最新的 Mixtral 8x7B 模型，支持英、法、意、德、西五种语言及代码，MT-Bench 得分为 8.3。
*   **Mistral-medium 端点**：质量最高的原型模型，支持上述五种语言及代码，MT-Bench 得分为 8.6。
*   **Mistral-embed 端点**：专为检索设计的嵌入模型，维度为 1024，MTEB 检索得分为 55.26。
*   **开发者工具**：API 兼容主流聊天接口规范，提供 Python 和 Javascript 客户端库，支持通过系统提示词进行高级别的内容审核。
*   **生态合作**：与 NVIDIA 合作整合了 TensorRT-LLM 和 Triton，使稀疏专家混合模型（MoE）兼容 TRT-LLM。
*   **访问权限**：已开放注册，平台容量正逐步提升，业务团队可协助加速获取访问权限。

**3) 风险/不足**
*   在平台稳定并过渡到完全自助服务的过程中，用户可能会遇到一些系统不完善的情况。

## 正文

我们的首批 AI 端点现已开放抢先体验。Mistral AI 将 La Plateforme 定位为开发者平台，用于在生产环境中部署、调用和定制其生成式模型。

本次 Beta 首批开放 4 个端点，包括 3 个聊天端点和 1 个嵌入端点，各自对应不同的性能与价格权衡。前两个端点 `mistral-tiny` 与 `mistral-small` 基于已发布的开源模型，`mistral-medium` 则是一个仍在部署环境中验证的更高性能原型模型。

### 模型训练与对齐

平台提供的是指令微调后的模型版本。Mistral AI 表示，这些模型先在公开网络数据上预训练，再基于标注数据进行指令微调，并结合高效微调、直接偏好优化等对齐方法，以提升模型的可控性和使用体验。

### 首批开放的端点

- `Mistral-tiny`：主打成本效率，当前对应 `Mistral 7B Instruct v0.2`。该端点仅支持英语，在 MT-Bench 上得分为 7.6，对应模型也开放下载。
- `Mistral-small`：基于 `Mixtral 8x7B`，支持英语、法语、意大利语、德语、西班牙语以及代码任务，在 MT-Bench 上得分为 8.3。
- `Mistral-medium`：当前质量最高的端点，提供一个原型模型，在标准基准测试中处于可用服务模型前列，同样支持上述多语言和代码任务，在 MT-Bench 上得分为 8.6。
- `Mistral-embed`：面向检索场景的嵌入端点，向量维度为 1024，在 MTEB 检索任务上得分为 55.26。

### API 与接入方式

Mistral AI 表示，其 API 遵循流行的聊天接口规范，并提供 Python 与 Javascript 客户端库。对于需要更严格输出控制的场景，端点支持通过系统提示词设置更高等级的内容约束。

随着平台容量逐步提升，开发者已可注册使用这套 API，业务团队也可协助评估需求并加速接入。官方同时提醒，平台仍处于从 Beta 走向完全自助服务的阶段，使用过程中可能会遇到尚未完善的部分。

### 基础设施合作

Mistral AI 还特别感谢 NVIDIA 在 TensorRT-LLM 与 Triton 整合中的支持。双方合作的目标之一，是让稀疏混合专家模型能够兼容 TRT-LLM。
