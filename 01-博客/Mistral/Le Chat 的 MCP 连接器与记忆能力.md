## 摘要

### 一句话总结
Mistral AI 的 Le Chat 推出了支持 20 多个企业平台的 MCP 连接器以及跨对话的“记忆”功能，旨在将 AI 助手转化为统一的数据处理与工作流执行平台。

### 关键要点
*   **丰富的企业连接器（测试版）**：发布 20 多个由 MCP 驱动的安全连接器，涵盖数据（如 Databricks、Snowflake）、生产力（如 Box、Notion、Asana）、开发（如 GitHub、Jira）、自动化（如 Zapier）和商业（如 Stripe）等核心类别。
*   **自定义扩展能力**：支持用户添加自定义的远程 MCP 服务器，以便查询、交叉引用并操作企业专属的系统和工作流。
*   **全新的记忆功能（测试版）**：能够在对话之间延续上下文，基于用户偏好和事实提供高度个性化的回复，并支持从 ChatGPT 快速导入记忆。
*   **精细的记忆控制**：记忆系统会自动保存重要内容并忽略敏感或短暂信息；用户拥有完全的控制权，可随时添加、编辑、更新或删除记忆条目。
*   **企业级安全与权限管理**：管理员可控制组织内连接器的使用权限，并支持代办身份验证（on-behalf authentication），确保用户仅能访问其拥有权限的数据。
*   **灵活的部署选项**：支持在移动设备、浏览器中运行，也可选择本地部署、私有云/公有云部署或作为 Mistral Cloud 中的全托管服务运行。
*   **免费开放**：所有的连接器和记忆功能目前均在 Le Chat 的免费计划中向所有用户提供。
*   **相关活动**：Mistral 将于 9 月 9 日举办线上 MCP 网络研讨会，并于 9 月 13-14 日在巴黎举办为期两天的 MCP 黑客松。

## 即插即用。

今天，我们在 Le Chat 中发布了 20 多个由 MCP 驱动的安全连接器，使您能够使用关键业务工具进行搜索、总结并执行操作。Le Chat 的连接器目录涵盖了核心类别，简化了您在聊天中集成工作流的方式。

全新面貌的连接器（Connectors）目录打通了直达企业工具的管道，将 Le Chat 变成了处理数据、文档和操作的统一平台。

- 数据：在 Databricks（即将推出）、Snowflake（即将推出）、Pinecone、Prisma Postgres 和 DeepWiki 中搜索和分析数据集。

- 生产力：在 Box 和 Notion 中协作处理团队文档，在 Asana 或 Monday.com 中快速建立项目看板，并在 Jira 和 Confluence 等 Atlassian 工具中进行分类处理。

- 开发：在 GitHub 中管理问题（issues）、拉取请求（pull requests）、代码库和代码分析；在 Linear 中创建任务，在 Sentry 中监控错误，并与 Cloudflare 开发平台集成。

- 自动化：通过 Zapier 扩展工作流，并在 Brevo 中扩展营销活动。

- 商业：访问并处理来自 PayPal、Plaid、Square 和 Stripe 的商户及支付数据。

- 自定义：添加您自己的 MCP 连接器以扩展覆盖范围，从而能够对您企业特有的系统和工作流进行查询、获取摘要并执行操作。

- 部署：在本地、您的云端或 Mistral Cloud 上运行，让您完全控制数据和工作流的存放位置。

### 连接器实战应用。

在 Databricks 中总结客户评论，然后在 Asana 中提交工单以解决首要问题。

在 GitHub 中审查未关闭的拉取请求，然后创建 Jira 问题以进行跟进，并在 Notion 中记录这些更改。

在 Box 中对比法律文档中的财务义务，然后将简明的摘要重新上传到 Box 中。

从 Jira 中总结活跃的问题，然后起草一份 Confluence 冲刺（sprint）概览页面供团队规划使用。

从 Stripe 检索业务支付洞察，然后将异常情况作为开发项目和任务记录在 Linear 中。

在我们的帮助中心了解更多关于连接器的信息。

### 连接任何 MCP 服务器。

对于其他所有需求，您现在可以连接到您选择的任何远程 MCP 服务器——即使它未列在连接器目录中——以对您技术栈中的任何工具进行查询、交叉引用和执行操作。

### 您的规则。由您掌控。

管理员用户可以放心地控制组织中哪些人可以使用哪些连接器，并支持代办身份验证（on-behalf authentication），确保用户只能访问他们被允许访问的数据。

以您的方式部署 Le Chat——自托管、在您的私有云或公有云中，或者作为 Mistral Cloud 中的全托管服务。请与我们的团队联系以了解企业级部署。

## 留住您的思绪。

Le Chat 中的记忆功能（Memories）可以在对话之间延续您的上下文，在需要时检索过去的洞察、决策和参考资料。它们能提供更相关的回复、为您量身定制的自适应建议，以及融入您具体工作细节的更丰富的答案——从而带来更快速、更相关且完全个性化的体验。

记忆功能在我们的准确性和可靠性评估中得分很高：保存重要内容，避免被禁止或敏感的推断，忽略短暂的内容，并在不产生幻觉的情况下检索正确的信息。

最重要的是，您始终拥有完全的控制权——随时添加、编辑、更新或删除任何条目，并拥有清晰的隐私设置和值得信赖的选择性记忆处理机制。
