## 正文
*本文整理自 Simon Willison 博客中收录的 Dimitris Papailiopoulos 关于使用 Claude Code 辅助研究的观点。*

### 零人力成本的“魔法盒”

对我而言，真正在智力上令人感兴趣的是另一件事。我现在拥有了一个类似“魔法盒”的东西：我只需把问题扔进去，就能几乎在“零人力成本”的情况下得到初步的答案。

### 探索新想法的范式转变

在此之前，我探索新想法的方式通常是这样的：
*   要么自己笨拙地拼凑出一些初步的验证内容；
*   要么让学生跑个简短的测试来寻找“信号”。如果信号存在，我们才会深入研究。

而现在，这种快速获取信号的步骤——也就是弄清楚一个问题是否具有实质性的研究价值——我完全可以在不占用任何人时间的情况下独立完成。现在，这仅仅是我、Claude Code 以及几天 GPU 算力之间的事情。

### 研究方式的未来

我不知道这从长远来看对我们的研究方式意味着什么，我认为目前也没有人能给出确切的答案。但是，**从提出问题到获得初步答案之间的距离，确实变得非常短了**。
