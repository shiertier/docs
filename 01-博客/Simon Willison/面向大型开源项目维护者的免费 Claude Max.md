## 摘要

**一句话摘要**
Anthropic 为符合条件的大型开源项目维护者提供为期六个月、价值每月 200 美元的免费 Claude Max 20x 订阅计划。

**关键要点**
- 计划内容：符合条件的申请者可免费获得原价 200 美元/月的 Claude Max 20x 套餐，有效期为六个月。
- 名额规模：该计划采用滚动审核制，名额上限高达 10000 个，显示出实质性的资源投放力度。
- 身份门槛：申请者必须是公共开源项目的主维护者或核心团队成员。
- 影响力指标：项目需满足 GitHub stars 超过 5000 个，或 NPM 包月下载量超过 100 万次。
- 活跃度要求：申请者在过去三个月内必须保持活跃的维护状态，而非仅有历史贡献。
- 基础设施支持：对于指标未达标但属于生态系统中关键基础设施的项目，Anthropic 明确表示也欢迎其提交申请。
- 核心目的：通过直接提供高额度 AI 资源，降低关键开源维护者使用前沿模型的门槛。

## 正文

Simon Willison 在这篇短文里转述了一项非常具体的支持计划：Anthropic 将把原价每月 200 美元的 Claude Max 20x 套餐，免费提供给大型开源项目维护者使用六个月。文章本身没有展开产品体验，而是把重点放在资格条件和计划规模上。

他摘录的官方条件有三项。第一，申请者需要是公共开源项目的 primary maintainer 或核心团队成员。第二，项目本身要达到足够大的公开影响力，例如仓库超过 5000 个 GitHub stars，或相关 NPM 包的月下载量超过 100 万。第三，申请者在最近三个月里仍需要保持活跃维护，而不是名义上的历史贡献者。

这项计划采用滚动审核，最多提供 10000 个名额。Anthropic 还补充说明：如果某个项目虽然没有足够醒目的 stars 或下载量，但在生态中实际承担了关键基础设施角色，也依然值得申请。Simon 特别注意到 10000 这个数字相当大，意味着这不是象征性的公关额度，而是一次面向开源维护者群体的真实资源投放。

从归档价值看，这篇文章记录的是 Claude 在开源生态中的一种支持方式：不是发布新模型或新工具，而是直接降低关键维护者获取高额度订阅的门槛，让他们能在长期维护工作中更现实地使用 Claude。
