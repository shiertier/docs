## 摘要

Tom Krcha 宣布推出 Pencil，这是一款面向 Claude Code 的本地无限设计画布，强调从设计到代码的直接衔接。

- 产品定位：Pencil 被描述为面向 Claude Code 的 INFINITE DESIGN CANVAS。
- 交互能力：强调超快 WebGL 画布、可编辑性和并行设计 agent。
- 工作流：可在本地与 Claude Code 联动，把设计转化为代码。
- 文件组织：设计文件可放入 Git 仓库，使用开放的 JSON `.pen` 格式。

## 正文

Tom Krcha 在 X 发文称：

> 很高兴发布 Pencil，
