# AI 画图技巧：提示词嵌入 HTML 让文字变红

## 文档信息
- 来源：https://baoyu.io/blog/ai-image-tip-red-text
- 发布日期：2025-04-03
- 作者：宝玉

## 摘要

**一句话总结**
本文介绍了在使用 Sora 或 GPT-4o 生成图片时，通过在提示词中嵌入 HTML 标签（如 `<span style="color:red">`）使指定文字变红的技巧，并提供了一个手绘风格信息图的完整提示词示例。

**关键要点**
*   **适用工具**：该文字变色技巧适用于 Sora 或 GPT-4o 的图像生成。
*   **核心语法**：在提示词中直接使用 HTML 代码包裹目标文字，例如 `<span style="color:red">「认知」</span>`，即可让该部分文字在生成的图片中显示为红色。
*   **示例图片规格**：提示词示例设定为 9:16 比例的竖版卡片，背景采用带有纸质肌理的米色或米白色。
*   **示例字体与排版**：标题要求使用红黑相间的大号毛笔草书，正文采用中文草书，整体分为 2 至 4 个清晰小节。
*   **示例视觉元素**：要求点缀简单、有趣的手绘插画或图标（如人物或象征符号），并预留足够的空白空间以保持视觉平衡。
*   **示例内容**：通过 HTML 标签将标题中的“「认知」”和“「圈子」”设置为红色，以突出“认知决定上限，圈子决定机会”的核心要点。

## 正文
使用 Sora 或者 GPT-4o 画图，可以对文字部分使用 HTML 代码，比如我要让某部分文字变成红色，可以写成 HTML 形式 `<span style="color:red">「认知」</span>`

参考提示词：

创作一张手绘风格的信息图卡片，比例为9:16竖版。卡片主题鲜明，背景为带有纸质肌理的米色或米白色，整体设计体现质朴、亲切的手绘美感。

卡片上方以红黑相间、对比鲜明的大号毛笔草书字体突出标题，吸引视觉焦点。文字内容均采用中文草书，整体布局分为2至4个清晰的小节，每节以简短、精炼的中文短语表达核心要点。字体保持草书流畅的韵律感，既清晰可读又富有艺术气息。

卡片中点缀简单、有趣的手绘插画或图标，例如人物或象征符号，以增强视觉吸引力，引发读者思考与共鸣。 整体布局注意视觉平衡，预留足够的空白空间，确保画面简洁明了，易于阅读和理解。
