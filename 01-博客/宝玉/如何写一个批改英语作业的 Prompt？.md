# 如何写一个批改英语作业的 Prompt？

## 文档信息
- 来源：https://baoyu.io/blog/write-english-homework-prompt
- 发布日期：2024-11-19
- 作者：宝玉

## 摘要

**1) 一句话总结**
本文通过类比培训实习老师的过程，详细讲解了如何编写用于批改英语作业的 AI Prompt，并提供了一个包含任务定义、执行步骤、输出格式和参考样例的标准模板。

**2) 关键要点**
*   **核心逻辑**：编写 Prompt 类似于指导实习生，需要明确告知任务内容、操作步骤、格式要求并提供参考样例。
*   **Prompt 四要素**：一个完整的批改作业 Prompt 应包含：1. 明确的任务；2. 具体的执行步骤；3. 规范的输出格式；4. 1-2个参考样例。
*   **迭代优化**：建议先用 1-2 篇文章让 AI 进行测试，根据输出结果是否满足要求来调整 Prompt，并针对常见错误补充提醒。
*   **模板任务设定**：要求 AI 找出语法、拼写、标点符号、用词选择等方面的错误，并给出明确建议。
*   **模板执行步骤**：规定 AI 需按照“阅读理解全文 -> 找出错误 -> 解释错误原因 -> 给出修改建议”的顺序执行。
*   **模板输出格式**：要求以列表形式呈现，每个条目必须包含“错误类型”、“原句”、“错误解释”和“修改建议”四个字段。
*   **模板注意事项**：强调反馈内容需简单易懂（符合学生理解水平），需逐一指出所有错误，且必须解释错误原因以便学生学习改正。

## 正文
问：想问问如何写一个批改英语作业的 Prompt？

答：对于这类任务，你可以想象一下，如果你要教刚上师范大学来实习的大学生去批改作业，你会怎么教他们做？

首先你得让他们知道要执行的任务是什么，然后你得告诉他们怎么一步步去做，批改的格式是什么，为了让他们给出的格式不至于千奇百怪，你还可能会把你以前批改过的作业给他们当样例参考。

教完后就让他们去批改作业试试，批改完了根据他们批改的结果再给出一些反馈，这样几次下来，这批实习生就可以把批改作业这件事做的不错了，以后来了新的实习生也一样可以按照你总结的流程、规则、格式来做。

让 AI 去批改作业也是类似的，你在 Prompt 里面要写清楚：

1.   任务是什么

2.   按照什么步骤来做

3.   输出格式是什么

4.   给1-2个样例作为参考

然后就可以发一两篇文章让 AI 去试试看，看是不是满足你要求，如果满足要求那很好，不满足要求再做出一些调整，根据常见的错误再给一些提醒

下面是一个Prompt参考：

```
批改学生提交的英语作业，指出语法、拼写、标点符号、用词选择等方面的错误，并给出明确的修改建议。

# Steps
1. 阅读学生提交的作业并理解全文内容。
2. 找出其中的错误，例如语法、拼写、标点符号、或用词不当等。
3. 对每个错误进行解释，说明为什么该用法不正确。
4. 给出对应的修改建议，帮助学生理解正确的写法。

# Output Format
请按照以下格式提供反馈：
- 错误类型：说明是哪种错误（如语法、拼写、标点等）。
- 原句：摘录出含有错误的句子。
- 错误解释：说明错误的原因，尽量简单易懂。
- 修改建议：给出如何修改的建议或正确的写法。

结果应以列表的形式呈现，类似如下：
<item>
    "错误类型": "语法错误",
    "原句": "She go to the store yesterday.",
    "错误解释": "谓语动词在过去时态下不应使用动词原形。",
    "修改建议": "She went to the store yesterday."
</item>
<item>
    "错误类型": "拼写错误",
    "原句": "It is a beatiful day.",
    "错误解释": "单词 'beatiful' 的拼写错误。",
    "修改建议": "It is a beautiful day."
</item>

# Notes
- 尽量保证反馈简单易懂，适合学生的理解水平。
- 如果有多个相同类型的错误，逐一指出。
- 不仅指出错误，还需解释原因，以便学生可以学习并改正类似错误。
```
