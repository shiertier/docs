# 如果是在国内的程序员普通的后端java开发，能怎么往llm的方向靠呢？

## 文档信息
- 来源：https://baoyu.io/blog/java-backend-developer-llm-transition
- 发布日期：2024-12-29
- 作者：宝玉

## 摘要

**一句话总结**
本文建议普通Java后端开发者通过使用AI辅助编程工具、开发LLM相关的业余项目（Side Project）以及将AI作为学习导师，以获取正反馈的方式逐步向LLM方向转型。

**关键要点**
* 转型LLM无需花费高价（如999元）报班，核心在于立即行动并建立快速的正反馈循环。
* 建议在深入LLM方向前，先学习并掌握AI辅助编程，以此作为熟悉LLM和提升效率的工具。
* 零成本起步：注册并利用Cursor等AI编程工具（如2周免费期），高频使用代码自动完成、对话、复杂Prompt生成及规则定制功能。
* 通过高频使用AI工具，直观感受并理解LLM的能力边界与实际价值。
* 开展业余项目（Side Project）：借助AI辅助编程，快速开发如翻译工具、AI聊天、文档对话等LLM相关的原型项目。
* 将AI作为专属导师：在写代码和做项目的过程中，随时让AI协助讲解知识、添加注释及解答疑问。
* 通过上述实践积累正反馈，最终打破单一语言（如Java）的局限，成长为熟练掌握LLM的程序员。

**风险/不足**
* 仅指望在公司日常工作中兼顾学习想学的新技术（如LLM）是非常困难的，不能以此作为转型的唯一途径。

## 正文
问：如果是在国内的程序员普通的后端java开发，能怎么往llm的方向靠呢？自己很想转，但不知道该怎么做比较好

答： 往 llm 方向靠没想的那么难，第一是有想法；第二是行动。行动了最好马上有正反馈，这样才能让自己坚持下去。想让自己马上有正反馈也不难，按照下面去做你一定会有正反馈，不需要交 ¥999 去报个班。

如果要学习 LLM 方向，我建议可以先学 AI 辅助编程再去学习 LLM 方向，磨刀不误砍柴工，一方面你可以借助 AI 编程熟悉 LLM，另外未来可以借助 AI 提升效率，以及遇到问题可以随时问 AI。

所以建议你可以：

1.   用 AI 编程，先注册个 Cursor 账号，有 2 周免费使用，0 成本起步，使劲的用，用它自动完成代码，用它聊天，用复杂的 Prompt 生成代码，写规则定制化，看它能做什么不能做什么，做着做着你就感受到 LLM 是什么，有什么价值。

2.   辅助 AI 编程做 LLM 相关的 Side Project。不要指望在公司能一边拿着工作一边学你想学的技术，这其实很难的，但是 Side Project 不一样，你可以做任何你想做的项目。比如你想做 LLM 相关的，那就可以去做起来，比如做个简单的翻译工具、AI 聊天、文档对话，也许以前是很难的事情，如果你已经上手了 AI 辅助编程，那么很快你就可以做一个原型出来。

3.   把 AI 当老师，写代码、做 Side Project 的过程中，不明白的，让 AI 给你讲解，给你加注释，有问题随时问它，你会发现你多了个随时解答问题的好老师。

有了上面的这些正反馈，你很快就会成为一个熟练掌握 LLM 的程序员，不再是一个 Java 或者某个领域的程序员。

行动起来，找到正反馈，坚持用，就这样！
