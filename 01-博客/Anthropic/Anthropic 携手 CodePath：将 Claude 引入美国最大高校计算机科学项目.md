# Anthropic 携手 CodePath：将 Claude 引入美国最大高校计算机科学项目

## 文档信息
- 来源：https://www.anthropic.com/news/anthropic-codepath-partnership

## 摘要
**1) 一句话总结**
Anthropic 宣布与美国最大的高校计算机科学教育机构 CodePath 达成合作，将 Claude 和 Claude Code 深度整合到其课程中，旨在为超过两万名来自不同背景的学生提供前沿的 AI 编程教育与实践机会。

**2) 关键要点**
*   **核心合作**：CodePath 将把 Claude 和 Claude Code 作为其课程和职业项目的核心，覆盖社区大学、州立学校以及传统黑人大学（HBCUs）的超两万名学生。
*   **受众背景**：CodePath 超过 40% 的学生来自年收入低于 5 万美元的家庭，合作旨在打破资源壁垒，提供行业验证的课程和职业网络。
*   **课程整合**：Claude 将被整合到“AI 工程基础”、“AI 工程应用”及“AI 开源顶点项目”等课程中，帮助学生使用 AI 工具为真实的开源项目做贡献。
*   **试点成果**：在 2025 年秋季的试点中，100 多名学生使用 Claude Code 成功为 GitLab、Puter 和 Dokploy 等开源项目做出了贡献，并有效克服了陌生编程语言的障碍。
*   **学分课程落地**：今年 1 月，霍华德大学推出重新设计的“人工智能入门”课程，这是 CodePath 的应用 AI 课程首次在大学中作为学分课程提供。
*   **联合研究**：Anthropic 与 CodePath 将开展公共研究，探索 AI 对编程教育和经济机会的影响，并分享实践经验。
*   **全球教育承诺**：Anthropic 正在全球扩展 AI 教育，包括为 180 万美国教师联合会（AFT）会员提供免费培训、在冰岛推出国家级 AI 教育试点，以及在卢旺达提供 Claude 学习助手。
*   **相关技术进展**：Anthropic 近期推出了扫描代码漏洞的 Claude Code Security（研究预览版），以及在编程和智能体方面性能领先的 Claude Sonnet 4.6。

**3) 风险与不足**
*   **加剧不平等的风险**：CodePath 首席执行官指出，AI 技术能大幅提升学习效率（两年学完四年内容），但如果只有部分群体能享受这种速度，将会加剧社会不平等。
*   **教育方法仍存不足**：在 AI 改变技术技能培养和职场准入方式的过程中，现有的教育实践方法仍有需要改进之处（双方将通过联合研究来探索和解决）。
*   **传统安全审查的漏洞**：传统的代码审查方法经常会遗漏安全问题（Anthropic 正通过推出 Claude Code Security 来辅助发现和修复这些遗漏）。

## 正文
随着人工智能正在重塑软件开发领域，Anthropic 宣布与美国最大的高校计算机科学教育机构 CodePath 达成合作，共同重新设计编程课程。CodePath 将把 Claude 和 Claude Code 作为其课程和职业项目的核心，为社区大学、州立学校以及传统黑人大学（HBCUs）的超过两万名学生提供在教育中使用前沿 AI 工具的机会。

CodePath 超过 40% 的学生来自年收入低于 5 万美元的家庭。该机构致力于为这些学生提供经过行业验证的课程，以及通常只有富裕学府学生才能接触到的职业网络。

### 合作内容与课程建设

CodePath 正在将 Claude 整合到其 AI 课程中——包括“AI 工程基础”、“AI 工程应用”以及“AI 开源顶点项目（Capstone）”。学生将学习使用 Claude Code 等工具进行开发，并为真实的开源项目做出贡献。

在 2025 年秋季的试点项目中，100 多名 CodePath 学生试用了 Claude Code，为 GitLab、Puter 和 Dokploy 等开源项目做出了贡献。德克萨斯理工大学计算机科学专业的学生 Laney Hood 表示：“Claude Code 在我的学习过程中发挥了重要作用，特别是当我对代码库中使用的编程语言（包括 TypeScript 和 Node.js）经验极少时。”

今年 1 月，霍华德大学（Howard University）宣布推出一门重新设计的“人工智能入门”课程。该课程由 CodePath 和 Thurgood Marshall 学院基金会合作开发，让学生获得 Claude 辅助软件开发的经验，为当前入门级工程师角色所需的工作做好准备。这也是 CodePath 的应用 AI 课程首次在大学中作为学分课程提供。

CodePath 联合创始人兼首席执行官 Michael Ellison 表示：“我们现在的技术可以在两年内教完过去需要四年才能学完的内容。但如果只有部分人能享受这种速度，只会加剧不平等。与 Anthropic 合作意味着我们的学生从第一天起就能在那些历史上被忽视的机构里，学习使用 Claude 进行开发。这将为我们的学生带来更好的结果，并从根本上改变‘谁能塑造 AI 经济’这一问题的答案。”

在课堂之外，Anthropic 和 CodePath 还将合作开展公共研究，探索 AI 如何改变编程教育以及经济机会的动态。双方将分享从学生、教育工作者和行业领袖那里学到的经验，探讨在 AI 改变人们建立技术技能和进入职场的方式时，哪些方法行之有效，哪些仍需改进。

### 对 AI 教育的广泛承诺

此次合作进一步推进了 Anthropic 将 AI 工具交到教育工作者和学生手中的努力。我们在全球范围内开展了多项教育合作：

*   **美国：** 我们与美国教师联合会（AFT）合作，为其全美 180 万名会员提供免费的 AI 培训。此外，我们签署了白宫的“对美国青年的承诺：投资 AI 教育”，承诺通过投资网络安全教育、总统 AI 挑战赛以及为教育工作者提供免费 AI 课程，在全国范围内扩展 AI 教育。
*   **冰岛：** 我们与冰岛教育和儿童部合作，推出了世界上首批国家级 AI 教育试点项目之一，让全国的教师都能使用 Claude。
*   **卢旺达：** 我们与卢旺达政府和 ALX 合作，为非洲数十万学生和年轻专业人士提供由 Claude 驱动的学习助手。

改变软件构建方式的工具不应仅仅局限于资源丰富的大学的学生。通过与 CodePath 的合作，我们将打破这一现状。

### 相关内容
