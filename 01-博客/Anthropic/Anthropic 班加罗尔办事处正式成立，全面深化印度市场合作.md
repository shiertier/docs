# Anthropic 班加罗尔办事处正式成立，全面深化印度市场合作

## 文档信息
- 来源：https://www.anthropic.com/news/bengaluru-office-partnerships-across-india

## 摘要
**1) 一句话总结**
Anthropic 正式启用班加罗尔办事处（其亚洲第二家），并宣布在企业、教育、农业及公共服务等领域达成多项合作，全面深化其在全球第二大市场印度的业务布局。

**2) 关键要点**
*   **市场地位与团队建设：** 印度是 Claude.ai 全球第二大市场（近半使用量集中在计算机和数学任务）；班加罗尔办事处由印度区董事总经理 Irina Ghose 领导，将专注于招募本土人才。
*   **本土语言优化：** 启动针对 10 种印度使用最广泛语言的数据优化计划，并联合多家机构构建农业和法律领域的性能评估测试，相关评估工具计划公开发布。
*   **企业端业务增长：** 自 2024 年 10 月以来，Anthropic 在印度的年化收入已翻倍。
*   **企业与初创应用案例：** 高知特（Cognizant）正向 35 万名员工部署 Claude；印度航空和 CRED 正在使用 Claude Code 提升软件交付效率；初创公司 Emergent 完全基于 Claude 构建，不到 5 个月实现 2500 万美元 ARR 及 200 万用户。
*   **教育领域合作：** 教育占印度 Claude 总使用量的 12%；与 Pratham 合作试点“随时测试机”（目前覆盖 1500 名学生，计划 2026 年扩至 100 所学校），并为 5000 多名辍学女性提供适配支持。
*   **公共部门与农业：** 联合 EkStep Foundation 等机构，将 Claude 应用于农业（OpenAgriNet 项目）、医疗（Noora Health 等）及司法服务（Adalat AI 推出基于 Claude 的 WhatsApp 全国法律帮助热线）。
*   **MCP 标准落地：** 印度统计和计划执行部（MoSPI）推出首个印度官方政府 MCP 服务器以开放国家统计数据；Swiggy 利用 MCP 实现通过 Claude 直接订购杂货和餐厅。
*   **其他产品与合作动态：** 推出具备前沿性能的 Claude Sonnet 4.6；发布 Claude Code Security 漏洞扫描功能（研究预览版）；与卢旺达政府签署医疗和教育 AI 合作谅解备忘录。

**3) 风险/缺口**
*   **AI 语言能力差距：** 明确指出当前 AI 模型在英语上的表现依然优于其他语言，存在本土语言流畅度和代表性数据的缺口。
*   **司法服务可及性缺口：** 印度存在 5000 万件未结案件，常规案件更新需数月，且高度依赖纸质文件和中介，存在严重的信息获取壁垒。
*   **传统安全方法缺陷：** 明确指出传统的网络安全扫描方法经常遗漏代码库中的安全问题（需通过引入 AI 辅助审查来弥补）。

## 正文
印度目前是 Claude.ai 的全球第二大市场，这里拥有全球技术最硬核的开发者社区之一。在印度，近一半的 Claude 使用量集中在计算机和数学任务上，包括构建应用程序、现代化升级系统以及交付生产级软件。

今天，随着班加罗尔办事处的正式启用，我们宣布在企业、教育和农业等多个领域达成全新合作伙伴关系，进一步深化我们对印度各行业的承诺。

Anthropic 印度区董事总经理 Irina Ghose 表示：“印度代表着全球最具潜力的机遇之一，能够将负责任的 AI 带来的红利惠及更广大的民众和企业。这里已经具备了顶尖的技术人才、规模化的数字基础设施，以及利用科技改善民生的成功经验。这正是确保这项技术能够触达最需要它的人群所必需的基石。”

### 提升十亿人口的本土语言能力

印度有超过十亿人使用十几种官方认可的语言之一，但目前 AI 模型在英语上的表现依然优于其他语言。六个月前，我们在全公司范围内启动了一项计划，通过在印度使用最广泛的 10 种语言（印地语、孟加拉语、马拉地语、泰卢固语、泰米尔语、旁遮普语、古吉拉特语、卡纳达语、马拉雅拉姆语和乌尔都语）中整理更高质量、更具代表性的训练数据，来缩小这一差距。这显著提升了我们的模型表现，我们也将继续致力于提高这些语言的流畅度。

目前，Anthropic 正在与 Karya 和 Collective Intelligence Project 合作，并联合 Digital Green 和 Adalat AI 等印度顶尖非营利组织的领域专家，共同构建针对农业和法律等本地相关任务的性能评估测试。这项工作将指导我们如何为印度语言使用者、印度核心应用场景以及使用 Claude 的企业改进未来的模型。我们计划将这些评估工具公开发布，供各界使用。

### 赋能企业、数字原生公司与初创企业

自 2024 年 10 月宣布扩张以来（注：原文为 2025 年，结合语境修复为合理时间），我们在印度的年化收入已经翻倍。基于 Claude 构建解决方案的组织类型广泛，从大型企业、数字原生公司到刚刚推出首款产品的初创企业，充分体现了这种增长的全面性。

为了支持不断增长的客户群，我们的印度团队将为企业客户、数字原生公司和初创企业提供应用型 AI 专业支持，帮助他们设计、构建和扩展量身定制的 Claude 驱动解决方案。

*   **Air India（印度航空）：** 正在使用 Claude Code 帮助开发者以更低的成本、更快的速度交付定制软件，这也是其在运营中全面推广代理式 AI（Agentic AI）战略的一部分。
*   **CRED：** 借助 Claude Code，实现了功能交付速度翻倍，测试覆盖率提升 10%。
*   **Cognizant（高知特）：** 正在向全球 35 万名员工部署 Claude，以现代化升级旧系统、加速软件开发，并支持其企业客户采用 AI。

在印度的初创企业中，同样充满活力：
*   **Razorpay：** 已将 AI 整合到全公司的风险系统、决策流程和日常运营中。
*   **Rocket：** 利用 Claude 让企业内的非技术团队能够在几分钟或几小时内（而非数周）构建出生产级应用和网站。
*   **Enterpret：** 其 AI 助手由 Claude 驱动，工程团队每天使用 Claude Code 进行开发，并且该初创公司已发布了 MCP 集成，将客户洞察直接引入 Claude。
*   **Emergent：** 这是一个由 AI 驱动的平台，允许任何人通过自然语言描述来构建软件。该平台完全基于 Claude 构建，在不到五个月的时间内就实现了 2500 万美元的年度经常性收入（ARR）并积累了 200 万用户。

### 惠及低收入社区的教育事业

教育和教学任务占印度 Claude.ai 总使用量的 12%。印度最大的教育非营利组织之一 Pratham 选择 Anthropic 作为其首个战略 AI 实验室合作伙伴，正是因为我们对安全性与教育严谨性有着共同的追求。

由 Claude 驱动的“随时测试机”（Anytime Testing Machine）目前正在 20 所学校的 1500 名学生中进行试点，并计划到 2026 年底扩展至 100 所学校。今年早些时候，该系统还针对 Pratham 的“第二次机会”项目（旨在帮助辍学女性重返课堂）中的 5000 多名学习者进行了适配，旨在通过帮助学生进行考试练习，为学习和认证创造灵活、可靠的途径。

此外，Anthropic 正在与 Central Square Foundation 合作，探索如何更有效地利用教育科技和 AI 来教育来自欠发达社区的儿童。Anthropic 将为开发 AI 工具（包括个性化辅导、教师培训解决方案和评估驱动教学）的组织提供技术专长、指导和 API 额度，目标是惠及印度各地更多的小学生。

### 推动人工智能融入公共部门与农业

印度在构建改善民生的互操作性数字公共基础设施方面有着良好的记录。Anthropic 正在与 EkStep Foundation 合作，探索 AI 如何在这些努力的基础上，在对印度最重要的领域产生人口规模的影响。

*   **农业领域：** 农业占印度经济的近六分之一，雇佣了近一半的劳动力。通过 OpenAgriNet 项目，我们正致力于部署 Claude，以扩大这一关键领域的专家知识获取渠道。
*   **医疗健康：** 我们正在展示 Claude Code 和 Cowork 如何在非营利组织内部发挥作用，包括为数百万家庭提供便捷健康指导的 Noora Health，以及将偏远社区患者与优质医疗服务连接起来的 Intelehealth。
*   **司法服务：** 印度有 5000 万件未结案件，常规的案件更新通常需要数月才能传达给诉讼当事人。获取案件信息通常需要反复前往法院或通过中介来处理纸质文件和法律术语。Anthropic 正在支持 Adalat AI 改善司法服务的可及性，并于今日推出全国性的 WhatsApp 帮助热线。该热线利用 Claude 提供即时的案件更新，并支持使用印度本土语言对法律文件进行翻译、摘要和交互式查询。

### 通过开源标准推动技术普及

Anthropic 创建了模型上下文协议（MCP），作为连接 AI 应用程序与外部系统的通用开源标准，并于近期将其捐赠给了 Linux 基金会。

在非营利组织 Bharat Digital 的支持下，印度统计和计划执行部（MoSPI）最近推出了首个印度官方政府 MCP 服务器，使 AI 系统的用户能够以开放和互操作的方式访问和查询权威的国家统计数据。在私营部门，Swiggy 利用 MCP 让用户能够直接通过 Claude 订购杂货和预订餐厅。

### 扩大在印度的业务版图

随着我们在印度业务的不断扩张，这些合作伙伴关系将在未来几个月和几年内持续深化。我们全新的班加罗尔办事处现已正式启用，这也是 Anthropic 继东京之后的亚洲第二家办事处。该办事处由企业和初创技术领域的资深领导者、印度区董事总经理 Irina Ghose 领导，将专注于在广泛的岗位上招募本土人才。

### 其他相关动态

*   **为防御者提供前沿的网络安全能力：** Claude Code Security 是一项内置于网页版 Claude Code 中的新功能，目前已进入有限的研究预览阶段。它能够扫描代码库中的安全漏洞，并提出针对性的软件补丁供人工审查，帮助团队发现和修复传统方法经常遗漏的安全问题。
*   **与卢旺达政府签署谅解备忘录：** Anthropic 与卢旺达政府签署了关于在医疗和教育领域应用 AI 的谅解备忘录。
*   **推出 Claude Sonnet 4.6：** Sonnet 4.6 在编程、代理任务和规模化专业工作方面均展现出了前沿的性能。
