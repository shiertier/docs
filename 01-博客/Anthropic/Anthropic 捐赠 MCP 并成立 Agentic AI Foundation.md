# Anthropic 捐赠 MCP 并成立 Agentic AI Foundation

## 文档信息

- 发布日期：2025-12-11
- 原文链接：https://www.anthropic.com/news/donating-the-model-context-protocol-and-establishing-of-the-agentic-ai-foundation

## 摘要

**1) 一句话总结**
Anthropic 将模型上下文协议（MCP）捐赠给新成立的智能体 AI 基金会（AAIF），该基金会由 Anthropic、Block 和 OpenAI 共同创立并隶属于 Linux 基金会，旨在确保智能体 AI 技术的开放、中立和社区驱动发展。

**2) 关键要点**
*   **基金会成立**：智能体 AI 基金会（AAIF）是 Linux 基金会旗下的定向基金，由 Anthropic、Block 和 OpenAI 共同创立，并获得 Google、Microsoft、AWS、Cloudflare 和 Bloomberg 的支持。
*   **创始项目**：MCP 与 Block 的 *goose* 以及 OpenAI 的 *AGENTS.md* 共同成为 AAIF 的首批创始项目。MCP 捐赠后，其原有的社区驱动和透明决策治理模式保持不变。
*   **MCP 普及数据**：目前生态系统中有超过 10,000 个活跃的公共 MCP 服务器；其 Python 和 TypeScript 官方 SDK 的月下载量超过 9,700 万次。
*   **跨平台与基础设施支持**：MCP 已被 ChatGPT、Cursor、Gemini、Microsoft Copilot 和 VS Code 等产品采用，并获得 AWS、Cloudflare、Google Cloud 和 Microsoft Azure 等云服务商的部署支持。
*   **技术与功能更新**：Claude 现拥有超 75 个由 MCP 驱动的连接器目录；API 新增了“工具搜索”和“编程式工具调用”功能以优化生产规模部署；11 月 25 日发布的规范引入了异步操作、无状态性、服务器身份和官方扩展等新特性。
*   **其他相关发布**：Anthropic 同期推出了具备前沿性能的 Claude Sonnet 4.6、用于扫描代码库漏洞的 Claude Code Security（有限研究预览版），并与卢旺达政府签署了关于医疗和教育领域 AI 应用的谅解备忘录。

## 正文

今天，我们将模型上下文协议 (Model Context Protocol, MCP) 捐赠给智能体 AI 基金会 (Agentic AI Foundation, AAIF)。该基金会是 Linux 基金会 (Linux Foundation) 旗下的一个定向基金，由 Anthropic、Block 和 OpenAI 共同创立，并获得了 Google、Microsoft、Amazon Web Services (AWS)、Cloudflare 和 Bloomberg 的支持。

## 模型上下文协议 Model Context Protocol

一年前，我们推出了 MCP，将其作为连接 AI 应用程序与外部系统的通用开放标准。自那时起，MCP 获得了惊人的普及率：

- **在整个生态系统中**：目前有超过 10,000 个活跃的公共 MCP 服务器，涵盖了从开发者工具到财富 500 强企业部署的各个方面；

- **跨平台**：MCP 已被 ChatGPT、Cursor、Gemini、Microsoft Copilot、Visual Studio Code 以及其他流行的 AI 产品所采用；

- **跨基础设施**：目前已存在企业级基础设施，AWS、Cloudflare、Google Cloud 和 Microsoft Azure 等提供商均提供了对 MCP 的部署支持。

我们正在继续投资于 MCP 的发展。Claude 现在拥有一个包含超过 75 个连接器（由 MCP 驱动）的目录，并且我们最近在 API 中推出了工具搜索 (Tool Search) 和编程式工具调用 (Programmatic Tool Calling) 功能，以帮助优化生产规模的 MCP 部署，高效处理数以千计的工具，并降低复杂智能体工作流中的延迟。MCP 现在拥有一个官方的、由社区驱动的注册表 (Registry)，用于发现可用的 MCP 服务器；此外，11 月 25 日发布的规范引入了许多新功能，包括异步操作、无状态性、服务器身份和官方扩展。MCP 还为所有主流编程语言提供了官方 SDK（软件开发工具包），其中 Python 和 TypeScript 的 SDK 每月下载量超过 9700 万次。自成立以来，我们一直致力于确保 MCP 保持开源、社区驱动和供应商中立。今天，我们通过将 MCP 捐赠给 Linux 基金会，进一步践行了这一承诺。

## Linux 基金会与智能体 AI 基金会

Linux 基金会是一家非营利组织，致力于通过中立的管理、社区建设和共享基础设施，促进可持续开源生态系统的发展。它在管理最关键和具有全球意义的开源项目（包括 Linux 内核、Kubernetes、Node.js 和 PyTorch）方面拥有数十年的经验。重要的是，Linux 基金会在促进开放协作和保持供应商中立性方面拥有良好的往绩。

智能体 AI 基金会 (AAIF) 是 Linux 基金会旗下的一个定向基金，由 Anthropic、Block 和 OpenAI 共同创立，并获得了 Google、Microsoft、AWS、Cloudflare 和 Bloomberg 的支持。AAIF 旨在通过战略投资、社区建设和共同开发开放标准，确保智能体 AI 以透明、协作和符合公众利益的方式发展。

## 捐赠模型上下文协议

Anthropic 正在将模型上下文协议捐赠给 Linux 基金会新成立的智能体 AI 基金会，它将与 Block 的 goose 和 OpenAI 的 AGENTS.md 一起成为创始项目。将这些项目以及未来的项目纳入 AAIF 旗下，将促进整个智能体 AI 生态系统的创新，并确保这些基础技术保持中立、开放和社区驱动。模型上下文协议的治理模式将保持不变：该项目的维护者将继续优先考虑社区的意见和透明的决策。

## MCP 的未来

开源软件对于构建安全、创新的智能体 AI 生态系统至关重要。今天向 Linux 基金会的捐赠表明了我们致力于确保 MCP 保持为中立、开放标准的承诺。我们很高兴能通过 AAIF 继续为 MCP 和其他智能体 AI 项目做出贡献。请访问 modelcontextprotocol.io 了解有关 MCP 的更多信息，并在此处参与 AAIF。
