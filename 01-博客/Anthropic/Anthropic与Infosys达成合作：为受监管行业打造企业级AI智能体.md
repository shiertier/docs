# Anthropic与Infosys达成合作：为受监管行业打造企业级AI智能体

## 文档信息
- 来源：https://www.anthropic.com/news/anthropic-infosys

## 摘要
**1) 一句话总结**
Anthropic与Infosys达成战略合作，将Claude系列模型与Infosys Topaz平台深度整合，共同为电信、金融等受监管行业开发和交付企业级智能体AI（Agentic AI）解决方案。

**2) 关键要点**
*   **技术整合**：Anthropic的Claude模型和Claude Code将与Infosys的AI优先服务平台Topaz相结合，在满足治理和透明度要求的前提下加速企业软件开发。
*   **印度市场布局**：印度是Claude.ai全球第二大市场（近半使用量用于构建应用和系统现代化），Infosys是Anthropic在印度扩大业务的首批重要合作伙伴。
*   **聚焦智能体AI**：双方将利用Claude Agent SDK等工具，构建能独立处理多步骤复杂任务（如处理索赔、合规审查）的AI智能体，并助力企业降低遗留系统迁移成本。
*   **行业定制方案**：
    *   *电信业*：网络运营现代化与智能自动化。
    *   *金融服务*：快速风险检测、自动化合规报告及个性化互动。
    *   *制造与工程*：加速产品设计与模拟，缩短研发周期。
    *   *软件开发与运营*：使用Claude Code加速代码开发，借助Claude Cowork自动化日常文档与审查工作。
*   **广泛的云支持**：Claude是目前唯一在Amazon Bedrock、Google Cloud Vertex AI和Microsoft Azure三大全球主流云平台上均可使用的前沿模型。
*   **安全工具预览**：推出Claude Code Security（有限研究预览版），可扫描代码库漏洞并提出人工审查补丁。
*   **其他最新动态**：Anthropic与卢旺达政府签署MOU（聚焦医疗与教育AI应用），并发布了在代码编写和智能体应用方面性能更强的新模型Claude Sonnet 4.6。

**3) 风险与缺口（Risks/Gaps）**
*   **落地鸿沟**：AI模型从演示阶段走向受监管行业的实际应用之间存在着巨大的鸿沟，必须依靠深厚的行业领域专长才能弥合。
*   **代码安全盲区**：在软件开发中，传统安全检测方法经常会遗漏代码库中的安全漏洞（需借助如Claude Code Security等新工具辅助发现）。

## 正文
Anthropic今日宣布与总部位于班加罗尔的下一代数字服务与咨询全球领导者Infosys达成合作。双方将共同为电信、金融服务、制造和软件开发等行业开发并交付企业级AI解决方案。

此次合作将Anthropic的Claude模型和Claude Code与Infosys Topaz（基于生成式和智能体AI技术的AI优先服务平台）深度整合，旨在帮助企业加速软件开发，并在满足受监管行业所需的治理和透明度要求下采用AI技术。

印度目前是Claude.ai的全球第二大市场，拥有高度活跃的开发者社区。在印度，近一半的Claude使用量集中在构建应用程序、系统现代化改造以及发布生产级软件上。Infosys正是Anthropic在印度扩大业务版图的首批重要合作伙伴之一。

### 结合领域专长，跨越行业落地鸿沟

Anthropic首席执行官兼联合创始人Dario Amodei指出，让AI模型从演示阶段走向受监管行业的实际应用，存在着巨大的鸿沟，而弥合这一鸿沟需要深厚的领域专长。Infosys在电信、金融服务和制造等重要行业恰好具备这种专业能力。其开发者已经在使用Claude Code加速工作，并为需要高精度、合规性和深厚领域知识的行业创建AI智能体。

Infosys首席执行官Salil Parekh也表示，AI不仅在改变商业模式，更在重新定义行业的运营和创新方式。双方的合作是推进企业级AI的战略性飞跃，旨在结合双方的专业优势，从利用智能风险管理和合规性推动金融服务现代化，到赋能工程企业以AI驱动设计与制造，全面加速全球企业的AI价值转化，使其更加智能、更具韧性且负责任。

### 为受监管行业构建智能体AI (Agentic AI)

此次合作的核心重点是“智能体AI”——这类系统不仅能回答问题，还能独立处理多步骤任务，如处理索赔、生成和测试代码或管理合规性审查。

借助Claude Agent SDK等工具，双方将帮助客户构建能够在漫长复杂的任务和流程中持续工作的AI智能体。此外，结合Infosys Topaz和Claude，企业可以更快速、轻松地进行遗留系统的迁移，降低老旧基础设施的更新成本。

双方将共同开发针对特定行业的定制化AI智能体：

*   **电信业**：帮助运营商实现网络运营现代化，简化客户生命周期管理，并改善服务交付，为这个全球运营最复杂且受高度监管的行业引入智能自动化。
*   **金融服务**：协助金融机构更快速地检测和评估风险，自动化合规报告，并提供更个性化的客户互动（例如根据客户的完整账户历史和市场状况量身定制财务建议）。
*   **制造与工程**：利用Claude加速产品设计和模拟，缩短研发周期，使工程师能够在投入生产前测试更多迭代版本。
*   **软件开发**：开发团队将使用Claude Code编写、测试和调试代码，加快从设计到生产的流程。
*   **企业运营**：借助Claude Cowork，团队可以自动化处理文档摘要、状态报告和审查周期等日常常规工作。

### 广泛的云平台支持

Claude是目前唯一在三大全球主流云服务平台（Amazon Bedrock、Google Cloud Vertex AI和Microsoft Azure）上均可使用的前沿模型。企业可通过Anthropic的企业服务页面进一步了解Claude及相关合作动态。

### 其他相关动态

*   **Claude Code Security**：作为Web端Claude Code的一项新功能，目前已进入有限研究预览阶段。它能够扫描代码库中的安全漏洞，并提出针对性的软件补丁供人工审查，帮助团队发现传统方法经常遗漏的安全问题。
*   **卢旺达政府合作**：Anthropic与卢旺达政府签署了谅解备忘录（MOU），致力于在医疗和教育领域应用AI技术。
*   **Claude Sonnet 4.6发布**：全新推出的Sonnet 4.6在代码编写、智能体应用以及大规模专业工作方面展现了前沿的性能。
