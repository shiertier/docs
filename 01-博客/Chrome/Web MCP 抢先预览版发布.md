## 摘要

Web MCP 现已推出抢先预览版，通过提供声明式和命令式 API，帮助网站向 AI 智能体公开结构化工具，从而实现更快速、可靠且精确的自动化交互。

- 核心目标：提供公开结构化工具的标准方式，使 AI 智能体能更高效、准确地在网站上执行操作。
- 交互机制：网站可通过定义工具直接告知智能体互动方式和位置，减少歧义并提升流程稳健性。
- API 形态：声明性 API 用于可在 HTML 表单中描述的标准操作；命令式 API 用于依赖 JavaScript 的复杂动态交互。
- 应用场景：覆盖客户支持、电商和旅游等场景，强调结构化数据下的自动化任务执行。
- 参与方式：原型试用面向抢先预览计划参与者开放，可通过 EPP 注册获取文档与演示。

## 正文
随着智能体网络的不断发展，我们希望帮助网站在 AI 智能体与其互动的方式中发挥积极作用。WebMCP 旨在提供一种用于公开结构化工具的标准方式，确保 AI 智能体能够以更高的速度、可靠性和精确度在您的网站上执行操作。

通过定义这些工具，您可以告知智能体如何以及在何处与您的网站互动，无论是预订航班、提交支持服务工单，还是浏览复杂数据。这种直接的沟通渠道消除了歧义，并实现了更快、更稳健的智能体工作流。

### 智能体网络的结构化互动

WebMCP 提出了两项新 API，允许浏览器智能体代表用户执行操作：

- 声明性 API：执行可直接在 HTML 表单中定义的标准操作。
- 命令式 API：执行需要 JavaScript 执行的复杂、动态交互。

这些 API 充当桥梁，使网站具备“智能体就绪”能力。与直接操作原始 DOM 相比，这种方式可以带来更可靠、性能更好的智能体工作流。

### 使用场景

可以设想一个能够自信且快速地为用户处理复杂任务的智能体。

- 客户支持：帮助用户创建详细的支持服务工单，智能体可自动填写必要技术细节。
- 电子商务：智能体可更容易找到目标商品、配置购物选项，并精确完成结账流程。
- 旅游：智能体可基于结构化数据进行搜索、筛选与预订，帮助用户更准确地获取目标航班。

### 加入抢先试用计划
