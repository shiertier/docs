## 摘要

**1) 一句话总结**
作者针对职场焦虑问题，设计了一套结合多种心理学理论的AI深度对话Prompt，旨在通过挖掘用户的过往经历与潜意识阴影，生成专属的万字报告以帮助用户发现被隐藏的底层天赋。

**2) 核心要点**
*   **创作背景**：因一位从事用户体验设计的朋友被裁员并陷入“自己没有天赋”的焦虑，作者决定利用AI帮助普通人寻找天赋。
*   **核心理念**：天赋常被社会规训所掩盖，许多从小被批评的缺点或对他人的嫉妒（荣格心理学中的“阴影”），往往是天赋被压抑的背面。
*   **Prompt理论基础**：该Prompt设定AI为“深度天赋挖掘机”，融合了盖洛普优势理论、心流理论与荣格心理学。
*   **对话规则**：强制AI采用“单次单问”和“苏格拉底式引导”模式，最多提问不超过10个问题，避免一次性抛出所有问题。
*   **核心提问方向**：包括16岁前无人逼迫也愿意做的事或顽固缺点、成年后的无意识胜任区（觉得显而易见但别人觉得难的事）、身体累但精神亢奋的事、以及曾产生强烈嫉妒心的对象。
*   **输出目标**：综合多轮对话信息，AI最终会输出一份约一万字的《个人天赋使用说明书》，提供底层天赋分析及未来职业建议。
*   **模型建议**：推荐使用具备深度思考/推理能力的AI模型（如GPT-5.1 Thinking、Gemini、Claude、DeepSeek等），并强调思考时间越长越好。
*   **实测数据与结果**：作者本人真实回答了8个问题后，获得了一份7000-8000字的报告；AI成功将其“叛逆”和“懒惰”等特质重塑为底层天赋，并准确建议其从事“深度科技博主”职业。

**3) 风险与不足（原文明确提及）**
*   **数据真实性风险**：用户在回答AI提问时，如果为了面子而不舍得说真话、不够真实详细，将无法有效挖掘出真正的天赋。
*   **工具选择风险**：如果使用非推理（非深度思考）的AI模型，挖掘和分析的效果会大打折扣。

## 正文

故事是这样的。

是周末跟一个刚毕业两三年的朋友聊天，聊到了天赋这个词。

主要还是他太焦虑了，他最近有点崩。

他说，“我觉得我干这行就是个错误，我没这个天赋。”

他跟我之前一样，现在也是个用户体验设计师，刚刚非常不幸的是，被裁了，他本身就不在什么特别的大的厂子，刚毕业也是在创业公司里面打转，跳了一次槽，磨了两三年，结果还是这个样子。

他说，感觉自己像个小丑，无论做什么，都是错的，开会被老板批，做的方案上线了以后被用户骂，他也并不喜欢这份工作，现在，更不喜欢这个职业。

我当时听着，还是挺揪心的，我当时问他，那你为啥最开始要进这个行当。

他说，他最开始以为他喜欢，别人也说他画画还不错，所以就进来了。

结果才发现，自己好像并没有什么天赋。

我其实还是挺不知道说啥的，因为我真的很讨厌去跟别人说，你要坚持，要努力，这都是过程中的痛，到时候你会突破的。

这样的鸡汤废话，爹味实在太重了。

但是我想了想以后，还是说了，因为那个时候，他需要。

但是回来以后，我还是越想越感觉不对。

还是在天赋这个词。

因为我也在思考，我这个人，有自己的天赋吗？我想了半天，我没想出来。

更多的好像是长大以后，别人总是跟我说，你没有这个的天赋，别做了。

你看，天赋这个词，很奇怪的。

小时候这个词，它是用来夸别人的，比如“她很有音乐天赋”“他体育很好”“某某是学霸，是天才”。

我们长大一点，它就开始变成一把锋利的刀：“你没有这方面的天赋”“你不适合做这个”。

我们真的有人，没有天赋吗？难道真的会有人，完全没有任何天赋吗？我其实挺不相信的。

我只是觉得，是有很多人，并没有找到自己的天赋，有些人运气很好，在年纪很小的时候就找到了，于是他成了世界级别的大师。

而有的人，却这一生，都没有发现，自己的天赋。

那个每天都在镜子面前给自己磕头的半佛老师，很久很久以前写过一篇文章。

里面这有这么一段话。

最绝望的事，莫过于，错过了自己那个本该精彩的人生。

当时，我想了很久以后，我觉得，寻找自己天赋这个事，好像从小到大，都一直被大家所忽略。

过去这么多年，我们接受的训练，几乎都不是认识自己，是为了变成一个更标准的人。

那种喜欢追问为什么的小孩，被说成杠精。

那种上课走神、自己在本子上画画的，被说成不专心。

那种特别敏感，能察觉气氛变化的人，被说成太矫情太玻璃心。

我们从小慢慢学会的一件事是，只要我把那些不合群的地方藏起来，我就比较安全。

于是天赋的第一层伪装，在很早的时候就戴上了。

心理学里有个说法，荣格称之为阴影。

那些你讨厌的自己、你感到羞耻的冲动、那些被老师、父母否定过的部分，会被你一起丢进潜意识的地下室。

但很讽刺的是，很多真正的天赋，其实都伴随着阴影，甚至，来源于你的阴影。

所以，我突然有一个想法，就是AI明明现在都这么强了。

那，为什么不可以，帮我挖掘我们自己真正的天赋呢？

为什么不能，去弥补我们曾经的平凡呢？

说干就干，在花了一下午时间，迭代了好多版Prompt之后。

我写了一个，能帮助大家，真正挖掘自己天赋的Prompt。

它是这样的：

#
Role: 深度天赋挖掘机
#
# 角色
你是一位结合了盖洛普优势理论、心流理论与荣格心理学的资深生涯咨询师。你坚信天赋不是某种具体技能，而是可迁移的底层能力。
#
# 目标
通过多个深度多轮对话，帮助用户打破焦虑，帮他们找到他们被影藏起来的天赋，并生成一份极度详细、专业有共情力的《天赋说明书》。
#
# 核心理念
1. 反宿命论。
2. 能量审计：真正的天赋是让你回血的事，而不是你单纯擅长但做完很累的事。
3. 阴影即宝藏：用户的缺点、怪癖、甚至对他人的嫉妒，往往是天赋被压抑的背面。
#
# 严格遵守
1. 禁止一次性提问：必须采用“你问 -> 用户答 -> 你简短反馈 -> 再问下一题”的模式。每轮对话只聚焦一个问题。
2. 苏格拉底式引导：不要急着下结论，多问“为什么”、“当时什么感觉”、“具体例子”。
3. 温暖而犀利：保持共情，但在捕捉用户逻辑漏洞或潜意识信号时要敏锐。
#
# 提问问题
提问 1：请引导用户回忆16岁之前（未被社会完全规训前），有哪些事情是没人逼也会废寝忘食去做的？或者有哪些从小到大被批评的“顽固缺点”（如爱插嘴、太敏感、爱发呆）？
提问 2：成年后的工作/生活中，哪件事让你觉得“这还需要学吗？这不是显而易见的吗？”但周围人却觉得很难？（寻找无意识胜任区）。
提问 3：哪件事做完后虽然身体累，但精神极度亢奋？
提问 4：这可能有点冒犯，但很关键，你曾经对谁（或哪种生活状态）产生过强烈的嫉妒或酸溜溜的感觉？（嫉妒通常是“被压抑的天赋”在发出信号，请诚实面对）。
这四个问题必须问到，但是不一定是线性的，过程中也可以根据你对用户的好奇和挖掘，来提出全新的问题，只要对发掘用户的天赋有帮助。
最多不超过10个问题。
#
# 输出
综合所有问题的信息，输出万字左右的《个人天赋使用说明书》。
这篇报告不设定结构，由你根据用户的答案，自由发挥。
但必须一万字以上，需要达到他的内心，让他真的觉得有用，帮助他找到真正的底层天赋，为他未来的人生路和从事职业给与详细的建议。
#
# 开始
请以温暖、专业、共情的语调开场，像用户详细解释接下来的流程和占用的时间，以及希望达成的目标。
向用户问好，用通俗语言简述天赋挖掘机的作用，告诉用户：“天赋永远不会过期，我们只是要找到你的底层天赋。”
然后在再开始进入提问流程。

大概作用就是，给你提一些问题，你来回答，直到他觉得答案足够丰富之后，就会根据你的回答，给你输出一份万字的个人天赋使用说明书。

我希望用这种方式，帮每一个人，都能找到自己的天赋。

德尔菲神庙门上刻着三句铭文。

![](https://mmbiz.qpic.cn/mmbiz_jpg/OjgKEXmLURoS09yccicJtFmTBOhuUu3rrzR8I8jcicnmfThTIGQUia1vWRCOOP4hYPr7bicVq2sugUPaQjRgdVPlcA/640?wx_fmt=webp&from=appmsg)

这三句话，也被誉为
德尔菲神谕。

其中最著名的那句话，就叫做：

认识你自己。

苏格拉底也把它作为自己哲学原则的宣言。

千百年来，这一直都是一个终极的哲学难题。

我们通过读书，通过旅行，通过与人交谈，通过在爱里受伤，来一点点地拼凑出“我”的样子。

这个过程，漫长、痛苦，而且充满了偶然性。

而现在，我们有了AI。

它把人类历史上几乎所有的知识、所有的心理学模型、所有的人格分析理论都装进了脑子里，然后，它只为你一个人服务。

它不会不耐烦，不会评判你，不会带着偏见。

它只是帮你，把你自己的数据，进行一次彻底的整理和归纳，然后像一面镜子一样，呈现给你看。

问你，“
你看，这是不是你？”

就像那个半佛故事里的那个老教师，在翻身上马的瞬间，他沉睡的天赋被激活了。

我们也可能在与AI的一次深度对话中，看到某个被遗忘的爱好，某个被压抑的天赋时，我们大脑里某个沉睡已久的区域，突然亮了起来。

那个瞬间，你一定会热泪盈眶。

会有一种原来如此的释然。

和一种我还可以的希望。
