# DHH：未来的 Agent 不该依赖 MCP 等特殊适配

## 文档信息

- 发布日期：2026-02-05
- 作者：David Heinemeier Hansson / @dhh
- 来源：
  - https://x.com/dhh/status/2019389155239948625?s=46

## 摘要

DHH 认为未来的 AI Agent 将像自动驾驶汽车一样，无需依赖 LIDAR 或 MCP 等特殊适配，即可直接与环境进行交互。

- 帖子由 David Heinemeier Hansson (@dhh) 于 2026 年 2 月 5 日发布，抓取时浏览量达 10.39 万。
- 帖子内附带了一个指向其个人博客（标题含 clankers-with-claws）的外部链接。
- DHH 指出当前 AI 领域的一切变化都非常迅速。
- 他预判并看好 AI Agent 能够直接与环境交互的未来发展方向。
- 他强调这种交互不需要（也不应依赖）LIDAR 或 MCP 这类特殊适配技术。

## 正文

DHH 在这条帖子中写道：

“现在 AI 领域的一切变化都非常快。但如果要预判未来方向，我更看好这样一个世界：Agent 像自动驾驶汽车一样，不需要 LIDAR 或 MCP 这类特殊适配，也能直接和环境交互。”
