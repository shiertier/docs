## 摘要

### 1) 一句话总结
SpotX 是一款专为 Windows 平台设计的 Spotify 官方桌面客户端补丁工具，主要用于全面屏蔽广告、定制主页内容、禁用数据收集以及控制版本更新。

### 2) 核心要点
* **环境要求**：支持 Windows 7-11，需依赖 PowerShell 5.1 及以上版本，且仅支持 Spotify 官方独立桌面版。
* **全面去广告**：可彻底屏蔽客户端内的所有横幅广告、视频广告和音频广告。
* **主页与功能定制**：支持从主页隐藏播客、单集和有声读物，并可解锁或修改部分原生实验性功能。
* **隐私与更新控制**：禁用客户端分析数据的发送，并提供阻止 Spotify 自动更新的选项。
* **多模式安装**：提供新主题模式、旧主题模式、全自动静默安装模式，以及专为已订阅用户设计的 Premium 专属模式（仅禁用播客音频广告）。
* **高级自定义**：支持通过高级参数进行灵活的自定义安装。
* **跨平台衍生**：官方另有提供针对 Mac 和 Linux 系统的衍生项目（SpotX-Bash）。
* **安装与卸载**：通过 PowerShell 脚本或批处理文件执行安装；可通过专用脚本或彻底重装官方客户端进行卸载。

### 3) 风险与不足
* **版本兼容性限制**：明确不支持通过 Microsoft Store（微软商店）下载的 Spotify 版本。
* **旧主题降级限制**：由于官方已移除旧版主题，使用旧主题模式必须强制降级并锁定在特定旧版本（如 1.2.13）。
* **合规与使用风险**：该工具涉及对官方客户端的修改与广告规避，官方声明其仅作为评估版本提供，用户需自行承担使用该工具可能带来的风险。

## 功能与定位
SpotX 是一个专门针对 Windows 平台 Spotify 官方桌面客户端的修改工具（Patcher）。其主要定位是通过修改客户端来屏蔽各类广告、定制主页内容、禁用数据收集，并解锁或调整部分实验性功能。

## 核心功能
* **全面去广告**：屏蔽客户端内的所有横幅广告、视频广告和音频广告。
* **主页内容定制**：提供从主页隐藏播客、单集和有声读物的选项。
* **更新控制**：可选择阻止 Spotify 客户端的自动更新。
* **隐私与功能调整**：禁用了客户端的分析数据发送，并修改了部分原生实验性功能。
* **高级参数支持**：支持通过高级参数进行灵活的自定义安装。

## 特色与差异点
* **多模式安装**：
  * **新主题模式**：激活全新的左右侧边栏及部分封面更改，包含所有实验性功能。
  * **旧主题模式**：强制安装特定旧版本（如 1.2.13）以保留旧版主题，并自动阻止更新。
  * **全自动模式**：无弹窗确认的静默安装，自动清理微软商店版、隐藏广告区块、激活静态歌词主题等。
  * **Premium 专属模式**：专为已订阅 Premium 的用户设计，不包含常规去广告功能，仅禁用播客中的音频广告。
* **跨平台支持**：虽然本项目针对 Windows，但官方提供了针对 Mac 和 Linux 的衍生项目（SpotX-Bash）。

## 使用方式概览
* **安装与更新**：通过在 PowerShell 中执行特定的安装脚本或运行批处理文件来完成。用户可根据需求选择常规安装（新/旧主题）、全自动安装或带参数的自定义安装。
* **卸载**：可通过运行专用的卸载脚本恢复，或通过完全卸载并重新安装官方 Spotify 客户端来移除该补丁。
*(注：由于该工具涉及客户端修改与广告规避，具体的安装脚本命令与下载直链已省略，请参阅官方仓库获取详细参数说明。)*

## 限制与注意事项
* **版本兼容性**：明确不支持通过 Microsoft Store（微软商店）下载的 Spotify 版本，必须使用官方独立桌面版。
* **旧主题限制**：由于 Spotify 在后续更新中移除了旧版主题，使用旧主题模式必须强制降级并锁定在旧版本。
* **风险提示（免责声明）**：SpotX 是一个修改官方客户端的工具，仅作为评估版本提供，用户需要自行承担使用该工具可能带来的风险。
