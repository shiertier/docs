## 摘要

### 1) 一句话总结
Social Analyzer 是一款基于 JavaScript 的开源情报（OSINT）工具，支持在 1000 多个网站上检索和深度分析社交媒体个人资料，提供 Web、CLI 和 API 三种形态以辅助安全调查与恶意活动溯源。

### 2) 核心要点
*   **检索范围广泛**：支持在 1000+ 社交媒体和网站中查找目标，允许按国家/地区（基于 Alexa 排名）或网站类型（如成人、音乐等）进行定向搜索。
*   **多层检测与低误报**：结合 OCR、HTTPS 库和 Webdriver 技术进行多层检测，并采用 0 到 100（否-也许-是）的评分机制量化匹配度，有效减少误报。
*   **深度分析与可视化**：支持提取元数据和模式，提供字符串排列组合、名称来源与相似度分析、有限的年龄估算等功能，并可使用 Ixora 生成力导向图进行可视化。
*   **专项检测模块**：针对 Facebook（电话号码、姓名或昵称）、Gmail 和 Google 账号提供专门的检测能力。
*   **多形态使用方式**：提供 Web 应用程序（官方推荐）、NodeJS/Python 命令行界面（CLI）、Python 代码直接集成，以及 Google Cloud Shell/repl.it 云端运行选项。
*   **高度可定制与集成**：支持自定义 User-Agent、代理、超时时间，支持并发搜索，分析结果可保存为 JSON 格式以便与其他 OSINT 工具集成。
*   **开源协议**：项目基于 AGPL-3.0 协议开源。

### 3) 风险与不足
*   **部署安全风险**：该工具未内置任何访问控制（Access Control）机制，仅定位为本地使用的安全项目，明确指出不应作为公开的在线服务部署。
*   **环境依赖限制**：部分功能（如网页截图）强制要求系统安装最新版本的 Chrome 浏览器；部分多层检测强依赖 Tesseract OCR 等外部系统组件。
*   **开源版本数据差异**：开源版本提供的检测数据库与部分执法机构使用的专属检测数据库不同。
*   **私有模块维护限制**：涉及以 `-private` 结尾或属于私有组的模块问题，禁止在 GitHub 上公开提交 Issue，必须直接联系项目作者。

## 功能与定位
Social Analyzer 是一款开源情报（OSINT）工具，旨在帮助用户在 1000 多个社交媒体和网站中分析和查找特定的个人资料。该项目提供 API、CLI（命令行界面）和 Web 应用程序三种使用形态，主要用于调查与网络欺凌、网络诱拐、网络跟踪和虚假信息传播等可疑或恶意活动相关的网络足迹。

## 典型使用场景
*   **安全调查与开源情报收集**：追踪特定目标在全网的社交媒体账号和公开信息。
*   **恶意活动溯源**：协助调查网络欺凌、跟踪或虚假信息传播者的背景资料。
*   **执法辅助**：在资源受限的国家或地区，部分执法机构使用该项目进行调查（注：执法机构使用的是与开源版本不同的专属检测数据库）。

## 核心功能
*   **跨平台广泛检索**：支持在 1000+ 网站中查找目标，可按国家/地区（基于 Alexa 排名）或网站类型（如成人、音乐等）进行定向搜索。
*   **多层检测机制**：结合 OCR（光学字符识别）、常规、高级和特殊检测技术，支持通过 HTTPS 库和 Webdriver 查找个人资料。
*   **元数据与模式提取**：提取目标元数据和模式，支持使用 Ixora 进行个人资料信息（元数据与模式）的可视化，并可生成力导向图（Force-directed Graph）。
*   **深度信息分析**：
    *   字符串与名称的排列组合分析。
    *   查找名称来源、名称相似度以及按语言分类的常用词。
    *   估算可能的个人资料/人物年龄（有限分析）。
*   **特殊目标检测**：针对 Facebook（电话号码、姓名或昵称）、Gmail 和 Google 账号提供专项检测模块。
*   **自动化与辅助工具**：支持多个人物资料的并发搜索（用于关联分析）、搜索引擎查询（Google/DuckDuckGo API）、自动过滤无关输出、提取网页截图及网站描述等。

## 特色与差异点
*   **低误报率的评分系统**：检测模块采用 0 到 100（否-也许-是）的评分机制来量化匹配度，旨在有效减少误报。
*   **灵活的集成能力**：提供 Python 和 NodeJS 版本的 CLI，支持作为 Python 对象直接导入代码中运行，分析结果可保存为 JSON 格式，便于与其他 OSINT 工具或信息流集成。
*   **丰富的自定义选项**：允许用户自定义 User-Agent、代理、超时时间和隐式等待时间，支持针对失败的个人资料进行重新检查，并可按匹配度（good, maybe, bad）过滤结果。

## 使用方式概览
*   **Web 应用程序（推荐）**：通过本地主机运行标准 Web 界面进行可视化操作。
*   **命令行界面（CLI）**：通过 NodeJS 或 Python 命令行执行，支持附加多种参数（如指定用户名、网站类型、输出格式、提取元数据等）。
*   **代码集成**：作为 Python 包或脚本运行，支持在 Python 项目中实例化 `SocialAnalyzer` 对象并调用。
*   **云端运行**：支持直接在 Google Cloud Shell 或 repl.it 中打开并运行。

## 限制与注意事项
*   **安全与部署限制**：该工具定位为本地使用的安全项目，未内置任何访问控制（Access Control）机制，不应作为公开的在线服务部署。
*   **环境依赖**：部分功能（如检测资料的截图功能）要求系统必须安装最新版本的 Chrome 浏览器；部分多层检测依赖 Tesseract OCR 等系统组件。
*   **私有模块支持**：对于以 `-private` 结尾或属于私有组的模块问题，需直接联系项目作者，不能在 GitHub 上公开提交 Issue。
*   **网站排除机制**：如果网站所有者希望将其网站从该项目的检测列表中排除，需主动联系项目作者。
