## 摘要

**1) 一句话总结**
SillyTavern 是一个专为高级用户设计的本地大语言模型（LLM）前端界面，支持统一接入多种文本、图像和语音生成 API，并提供高度可定制的提示词控制与丰富的功能扩展。

**2) 关键要点**
* **项目定位**：一款免费开源的本地安装界面，不提供任何在线托管服务，也绝不在后台通过程序追踪任何用户数据。
* **广泛的 API 支持**：支持众多 LLM API（如 OpenAI、Claude、Mistral、KoboldAI/CPP、OpenRouter 等）、图像生成引擎（Automatic1111、ComfyUI）以及 TTS 语音模型。
* **核心功能**：提供移动端友好布局、视觉小说模式、世界信息（lorebooks）、UI 自定义、自动翻译以及极其丰富的提示词选项。
* **高扩展性**：支持通过第三方扩展插件实现功能的无限增长。
* **硬件要求**：运行前端界面仅需 NodeJS 18 或更高版本；若需在本地进行 LLM 推理，官方建议使用至少配备 6GB 显存的 NVIDIA 3000 系列显卡。
* **开源与发展**：基于 GNU AGPL v3.0 协议开源，自 2023 年 2 月作为 TavernAI 1.2.8 的分支独立开发以来，已积累超过 200 名贡献者。
* **多平台支持**：提供详细的文档，支持在 Windows、MacOS/Linux、Android (Termux) 以及 Docker 环境下安装。

**3) 风险/缺口**
* 项目在基本信息中被归类为“风险边界与合规”类别，且带有“风险标记：1”。
* 官方明确指出该项目旨在赋予用户最大程度的控制权，因此存在“陡峭的学习曲线（steep learning curve）”。
* 软件按“原样”分发，明确声明不提供任何形式的担保（包括但不限于适销性或特定用途适用性的暗示保证）。
