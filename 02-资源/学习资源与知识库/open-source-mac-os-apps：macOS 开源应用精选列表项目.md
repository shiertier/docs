## 摘要

**一句话总结**
该项目是一个收集了689款macOS平台开源应用的精选列表，旨在帮助用户发现免费工具并鼓励开发者参与开源生态贡献。

**核心要点**
* **项目热度与协议**：在GitHub上拥有超4.7万Stars和2.4千Forks，采用CC0-1.0开源协议。
* **海量收录与分类**：共收录689款macOS开源应用（含原生与跨平台），细分为音频、开发工具、效率软件等49个类别。
* **详尽的项目信息**：每个应用条目均包含功能简介、链接，并附带展示最新版本、Stars数量、最后提交时间和开源协议的徽章。
* **可视化预览**：支持通过折叠面板（Details）直接在列表中查看应用界面的截图。
* **技术栈透明**：详细统计并标注了开发语言，排名前三的语言为Swift（304款）、Objective-C（137款）和JavaScript（113款）。
* **直观的排版设计**：为每种编程语言设计了专属图标，列表排版图文并茂且信息密度高。
* **社区驱动与订阅**：提供专属Telegram频道用于推送最新或热门应用，开发者可通过阅读 `CONTRIBUTING.md` 提交新的开源项目。

## 功能与定位

这是一个专门收集和整理 macOS 平台开源软件的精选列表（Awesome List）。该项目旨在帮助用户发现免费的开源 Mac 应用（包含原生应用与跨平台应用），同时鼓励开发者参与开源项目的贡献。

## 典型使用场景

* 寻找特定类别的 macOS 免费开源替代软件（如音频处理、开发工具、效率软件等）。
* 开发者寻找基于特定编程语言（如 Swift、Objective-C）的开源项目，用于源码学习或参与贡献。
* 跟踪和发现 macOS 生态中最新或最受欢迎的开源工具。

## 核心功能

* **海量应用收录**：目前共收录了 689 款 macOS 开源应用。
* **细致的分类体系**：将应用划分为 49 个类别，涵盖音频、备份、浏览器、聊天、加密货币、数据库、开发工具、编辑器、图形、效率工具、实用工具等。
* **详尽的项目信息**：列表中的每款应用均提供功能简介、开发语言标识、官网或仓库链接，并附带徽章（展示最新版本、Stars 数量、最后提交时间和开源协议）。
* **可视化预览**：支持通过折叠面板（Details）直接在列表中查看应用界面的截图。

## 特色与差异点

* **技术栈透明与统计**：详细统计并标注了应用的开发语言。排名前列的语言包括 Swift（304款）、Objective-C（137款）、JavaScript（113款）、C++（59款）和 TypeScript（42款）。
* **图文并茂的排版**：为每种编程语言设计了专属图标，列表排版直观，信息密度高。
* **社区驱动**：提供专属的 Telegram 频道，用于向订阅者推送最新或热门的开源应用。

## 使用方式概览

* 在项目 README 的目录（Contents）中点击对应分类，快速跳转至相关应用列表。
* 通过应用条目中的链接直接访问项目主页或下载页面。
* 订阅官方 Telegram 频道以获取新应用更新。
* 开发者可通过阅读 `CONTRIBUTING.md` 提交新的开源应用建议。
