## 摘要

**一句话总结**
doocs/md 是一款专为微信公众号排版设计的开源 Markdown 编辑器，支持丰富的扩展语法、多图床集成、AI 辅助创作以及多种便捷的私有化部署方式。

**核心要点**
*   **产品定位**：基于 Vue 开发的开源编辑器，可将 Markdown 文档即时、自动渲染为样式优雅的微信图文，解决传统排版样式繁杂的痛点。
*   **丰富的语法支持**：除基础 Markdown 外，支持数学公式、Mermaid/PlantUML 图表、GFM 警告块以及 Ruby 注音扩展（如 `[文字]{注音}`）。
*   **多图床集成**：内置支持 10 余种图床服务（包括 GitHub、阿里云 OSS、腾讯云 COS、微信公众号、MinIO 等），并支持自定义上传。
*   **AI 智能辅助**：集成 DeepSeek、OpenAI、通义千问、腾讯混元等主流 AI 模型，帮助用户高效完成内容创作与排版。
*   **编辑与管理功能**：提供代码块高亮主题，支持草稿自动保存、本地内容管理以及文件的导入与导出。
*   **高度定制化**：允许用户灵活自定义主题色和 CSS 样式，以控制最终图文展示效果。
*   **多端与多形态**：提供在线网页版，支持打包为 Chrome/Firefox 浏览器扩展、uTools 插件，以及部署至 Cloudflare Workers。
*   **私有化部署**：提供 npm CLI 工具（`@doocs/md-cli`）和 Docker 镜像（`doocs/md:latest`），支持通过极简命令快速搭建本地或私有实例。

**风险与限制**
*   **浏览器兼容性限制**：官方明确推荐使用 Chrome 浏览器访问在线编辑器，以获得最佳的使用与渲染效果。

## 功能与定位

doocs/md 是一款专为微信公众号排版设计的开源 Markdown 编辑器（基于 Vue 开发）。它能够将 Markdown 文档自动、即时地渲染为样式简洁优雅的微信图文，旨在解决传统微信排版过程中样式繁杂、需要反复调整的痛点。

## 典型使用场景

- **微信公众号图文排版**：通过编写基础 Markdown 语法，快速生成美观大方的微信图文。
- **AI 辅助创作与排版**：借助内置的 AI 助手，用户即使不熟悉 Markdown 语法也能高效完成内容创作与排版。

## 核心功能

- **完整 Markdown 与扩展语法**：支持基础语法、数学公式、Mermaid 图表、PlantUML 图表、GFM 警告块，以及 Ruby 注音扩展（如 `[文字]{注音}` 格式）。
- **多图床集成**：内置支持 10 余种图床服务，包括 GitHub、阿里云 OSS、腾讯云 COS、七牛云、MinIO、S3 协议兼容存储、微信公众号、Cloudflare R2、又拍云、Telegram、Cloudinary，并支持自定义上传。
- **AI 智能辅助**：集成 DeepSeek、OpenAI、通义千问、腾讯混元、火山方舟、302.AI 等主流 AI 模型，辅助内容创作。
- **编辑与内容管理**：提供丰富的代码块高亮主题，支持草稿自动保存、本地内容管理以及文件的便捷导入与导出。

## 特色与差异点

- **高度定制化**：允许用户自定义主题色和 CSS 样式，灵活定制最终的图文展示效果。
- **多端与多形态支持**：除了提供在线网页版，项目还支持打包为 Chrome/Firefox 浏览器扩展、uTools 插件，以及部署至 Cloudflare Workers。
- **私有化部署便捷**：提供 npm CLI 工具和 Docker 镜像，支持开发者通过极简命令快速搭建完全私有化的运行实例。

## 使用方式概览

- **在线使用**：直接访问官方提供的网页端编辑器。
- **插件模式**：可编译并加载为 Chrome/Firefox 浏览器扩展或 uTools 插件。
- **私有化部署（npm）**：通过全局安装 `@doocs/md-cli` 并运行 `md-cli` 启动本地服务（可自定义端口）。
- **私有化部署（Docker）**：通过运行 `doocs/md:latest` 镜像快速启动容器实例。

## 限制与注意事项

- 官方推荐使用 Chrome 浏览器访问在线编辑器，以获得最佳的使用与渲染效果。
