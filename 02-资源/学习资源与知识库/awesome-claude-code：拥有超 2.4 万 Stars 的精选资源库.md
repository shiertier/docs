## 摘要
**1) 一句话总结**
`hesreallyhim/awesome-claude-code` 是一个拥有超 2.4 万 Stars 的精选资源库，专门汇集了用于增强 Anthropic Claude Code CLI 功能的各类 Agent 技能、钩子、斜杠命令、工作流及插件。

**2) 关键要点**
*   **项目热度与基础信息**：该仓库备受欢迎，拥有 24,576 个 Stars 和 1,476 个 Forks，主要涉及 Python 语言，旨在促进社区分享 Claude Code 的最佳实践。
*   **资源分类全面**：目录涵盖了 Agent 技能（Agent Skills）、工作流与知识指南、工具（Tooling）、状态栏、钩子（Hooks）、斜杠命令（Slash-Commands）、CLAUDE.md 文件及官方文档镜像等。
*   **近期新增亮点工具**：收录了 `claude-rules-doctor`（检测失效的规则文件）、`ClaudeCTX`（一键切换配置）、`AgentSys`（自动化工作流系统）以及 `Auto-Claude`（自主多智能体编码框架）等最新实用工具。
*   **丰富的 Agent 技能库**：收集了针对特定任务的专业配置和脚本，覆盖 DevOps 部署、全栈开发、安全审计（如 Trail of Bits 提供的漏洞检测技能）、上下文工程及非虚构类书籍出版等领域。
*   **工作流与知识指南**：汇总了大量教程与架构模式，包括《Claude Code 终极指南》、Agentic 工作流模式（如子智能体编排、并行工具调用）、系统提示词解析以及端到端的项目管理（PM）工作流。
*   **官方文档与系统提示词镜像**：包含多个 Claude Code 官方文档的实时镜像仓库，以及针对不同版本更新的内置系统提示词（System Prompts）解析资源。
