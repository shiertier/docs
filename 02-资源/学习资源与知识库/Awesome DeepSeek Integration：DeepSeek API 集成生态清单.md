## 摘要
该项目是由 DeepSeek 官方维护的精选资源列表，专门收集和分类整理将 DeepSeek API 集成到各类软件、工具和开发框架中的案例。

## 功能与定位
`awesome-deepseek-integration` 是一个生态索引型仓库，核心目标是集中展示 DeepSeek API 在不同软件形态中的可用集成路径，帮助开发者与用户快速找到可复用方案。

## 典型使用场景
- 开发者寻找可直接参考的 DeepSeek 接入案例。
- 团队评估不同框架/客户端对 DeepSeek 的支持情况。
- 个人用户筛选已支持 DeepSeek 的桌面应用、插件或扩展工具。

## 核心功能
- 以分类目录维护集成清单。
- 覆盖应用、Agent 框架、RAG 框架、编辑器扩展、浏览器扩展、Providers 等多类入口。
- 提供多语言 README，降低跨语种阅读门槛。
