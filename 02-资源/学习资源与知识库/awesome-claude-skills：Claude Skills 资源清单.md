## 摘要
**1) 一句话总结**
`ComposioHQ/awesome-claude-skills` 是一个精选的 Claude 技能（Skills）、资源和工具集合，旨在通过自定义工作流全面提升 Claude.ai、Claude Code 和 Claude API 的生产力与自动化能力。

**2) 关键要点**
* **项目热度与定位**：该项目主要基于 Python，拥有超过 3.6 万 Stars 和 3500+ Forks，是极受欢迎的 Claude 学习资源与知识库。
* **核心概念（Claude Skills）**：提供可自定义的工作流，指导 Claude 跨平台以可重复、标准化的方式执行特定任务，突破了纯文本生成的限制。
* **强大的应用连接能力**：通过底层的 Composio 技术提供 `connect-apps` 插件，使 Claude 能够连接并真实操作 500+ 外部应用（如发送 Gmail、创建 GitHub Issue、发布 Slack 消息等），并预置了 78 个 SaaS 应用的自动化工作流。
* **文档与数据处理**：支持对 Word、PDF、PPT、Excel、CSV 等格式文件进行提取、编辑、分析和可视化，并支持 Markdown 转 EPUB 电子书。
* **开发与代码工具**：涵盖前端构建（artifacts-builder）、AWS 开发、MCP 服务器构建、Playwright 浏览器自动化、LangSmith 调试、iOS 模拟器交互及多种测试驱动开发（TDD）工具。
* **业务与营销支持**：提供品牌指南应用、竞品广告提取、域名头脑风暴、内部通讯撰写及潜在客户研究等商业技能。
* **多媒体与创意生成**：支持 Canvas 设计、Gemini 图像生成、图像画质增强、Slack GIF 制作及 YouTube 视频下载与字幕提取等功能。
* **安全与系统管理**：包含计算机数字取证、安全文件删除、元数据提取及使用 Sigma 规则进行安全威胁狩猎等系统级技能。
