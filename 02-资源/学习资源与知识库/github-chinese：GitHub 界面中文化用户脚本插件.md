## 摘要
**1) 一句话总结**
`maboloshi/github-chinese` 是一个广受欢迎的开源 JavaScript 用户脚本插件，旨在通过智能正则匹配、机器翻译和持续更新的词库，将 GitHub 界面全面中文化。

**2) 关键要点**
*   **项目基础**：采用 JavaScript 编写，遵循 GPL-3.0 开源协议，目前已获得超过 20,000 个 Stars 和 1,100+ 次 Forks。
*   **核心功能**：支持 GitHub 界面元素全面中文化、智能正则匹配、项目描述人机翻译，以及根据中文环境自动本地化时间元素。
*   **环境兼容**：兼容 Chrome/Chromium、Safari、Firefox 等主流浏览器内核，需配合 Tampermonkey、Violentmonkey、Stay 等脚本管理器使用。
*   **版本发布**：提供“开发版”（GitHub 源，每周五自动更新词库）和“稳定版”（GreasyFork 源，每周一同步词库）两种选择。
*   **最新更新**：最新版本 v1.9.3（2024-08-18）引入了全局缓存模式，优化了突变翻译处理规则（如 `characterDataPage`），并将翻译引擎升级至讯飞听见 v2.0。
*   **本地调试**：支持开发者通过修改 `@require` 路径并开启脚本管理器的“允许访问文件网址”权限来进行本地词库调试。
*   **未来规划**：官方待办事项包括添加 GitHub 专用名词解释、整理 Git/GitHub 学习资料以及进一步完善文档翻译。

**3) 风险/缺口**
*   **盗版/侵权风险**：官方明确警告，本项目从未发布至 GitCode 平台，若在该平台发现此项目属于未经授权的搬运。
*   **浏览器限制问题**：由于 Chrome 切换到 Manifest V3，基于 Chrome/Chromium 内核的浏览器用户必须手动开启扩展管理中的“开发者模式”和“允许运行用户脚本”才能正常使用。
