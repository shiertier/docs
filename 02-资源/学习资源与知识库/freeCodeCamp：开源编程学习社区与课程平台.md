## 摘要
### 一句话总结
freeCodeCamp 是一个由非营利组织运营的开源代码库和免费学习平台，提供全栈开发、机器学习及多语言认证课程，旨在帮助大众免费学习编程和计算机科学。

### 核心要点
*   **项目背景与影响力**：由 501(c)(3) 非营利慈善机构运营，致力于帮助成年人转型进入科技行业，目前已帮助超过 10 万人获得首份开发者工作。
*   **极高的社区关注度**：项目主要使用 TypeScript 开发，在 GitHub 上拥有超过 43.7 万 Stars 和 4.3 万 Forks，是全球最受欢迎的开源项目之一。
*   **免费技术课程与认证**：提供完全免费、自定进度的全栈 Web 开发和机器学习课程。涵盖响应式网页设计、JavaScript、前端库、Python、关系型数据库和后端 API 等认证。
*   **认证获取要求**：开发者认证需要完成交互式课程，提交 5 个必做项目并通过最终考试；获得的认证可永久保留并支持在 LinkedIn 或简历中验证。
*   **语言能力认证**：除技术课程外，还提供基于国际熟练度标准的免费语言认证（包括针对开发者的英语、专业西班牙语和专业中文的 Beta 版）。
*   **学术诚信政策**：平台严格执行学术诚信，若发现用户存在明确的抄袭行为（未注明出处提交他人代码），将撤销其认证并进行封禁。
*   **丰富的学习生态**：除主站平台外，社区还提供问答论坛、YouTube 免费课程频道、技术文章出版物以及 Discord 交流服务器。
*   **面试准备资源**：内置 The Odin Project（freeCodeCamp 混音版）、编程面试准备、Project Euler、Rosetta Code 以及与微软合作的 C# 基础认证。
*   **开源与版权协议**：项目采用混合许可模式，软件代码遵循 BSD 3-Clause 开源协议，而 `/curriculum` 目录下的课程学习资源则受 freeCodeCamp.org 版权保护。
