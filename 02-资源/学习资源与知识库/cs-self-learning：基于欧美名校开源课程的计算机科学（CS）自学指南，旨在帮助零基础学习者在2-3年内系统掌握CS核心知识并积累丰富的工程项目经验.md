## 摘要

**1) 一句话总结**
`PKUFlyingPig/cs-self-learning` 是一个基于欧美名校开源课程的计算机科学（CS）自学指南，旨在帮助零基础学习者在2-3年内系统掌握CS核心知识并积累丰富的工程项目经验。

**2) 关键要点**
* **项目定位**：提供系统化的计算机自学路径与优质资源库，官方在线阅读网站为 `csdiy.wiki`。
* **学习目标**：规划了2-3年的学习路径，涵盖数学基础、主流编程语言（C/C++/Java/Python/Go/Rust等）以及算法、操作系统、网络、人工智能、分布式等全面领域。
* **项目热度**：在GitHub上极受欢迎，目前已积累超过 71,000 个 Stars 和 7,800 余次 Forks。
* **开源协议**：项目贡献者编写的内容遵循 MIT License，引用的外部课程、书籍及视频资源则遵循原作者许可。
* **多语言支持**：指南同时提供中文和英文版本，提交贡献时需同步提供对应的英文翻译。
* **社区共建**：鼓励开发者通过 Pull Request 补充新课程（提供模板）、推荐书籍或修正错误，并要求遵循特定的中英混合排版规范。
* **交流互动**：网站支持页面评论功能，学习者可借此发布学习目标并自行组建或加入（QQ/微信）课程交流群。
* **商业赞助**：项目 README 中包含来自 AutoMQ 的赞助信息及相关工程实践推广。
