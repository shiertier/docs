## 摘要

**1) 一句话总结**
这是由微软开源的面向初学者的 Web 开发教程，提供为期12周、共24节课的综合课程，通过丰富的项目实践教授 HTML、CSS 和 JavaScript 基础知识。

**2) 关键要点**
* **课程规模与结构**：包含12周共24节课，采用基于项目的教学法（如开发在线玻璃容器、打字游戏等），主要编程语言为 JavaScript。
* **丰富的教学环节**：每节课均包含课前/课后测验（共48个测验）、书面教程、步骤指南、知识检查、挑战任务和课后作业。
* **开发环境友好**：支持在浏览器中通过 GitHub Codespaces 免安装运行，或在本地使用 Visual Studio Code（推荐搭配 Live Server 和 Copilot 插件）进行开发。
* **多语言支持**：通过 GitHub Action 自动维护 50 多种语言的翻译版本（包含简体/繁体中文）。
* **引入 AI 前沿技术**：课程最新加入了 GitHub Copilot Agent 模式挑战任务、生成式 AI 助手项目，并关联了全新的 JavaScript 生成式 AI 课程。
* **受众与社区支持**：为学生提供专属资源中心（含免费证书兑换券），为教师提供教学建议指南，并设有 Azure AI Foundry Discord 社区供学习者交流。
* **项目热度**：采用 MIT 开源协议，目前已获得超过 9.5 万 Stars 和 1.5 万 Forks，是极受欢迎的学习资源库。

**3) 风险/不足**
* **下载体积过大**：仓库中包含 50 多种语言的翻译文件，这会显著增加完整克隆时的下载体积（官方建议使用 `sparse-checkout` 稀疏检出命令来跳过翻译文件，以加快下载速度）。
