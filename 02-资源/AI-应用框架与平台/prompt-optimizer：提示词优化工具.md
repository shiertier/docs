## 摘要
**1) 一句话总结**
Prompt Optimizer 是一款基于 TypeScript 开发的开源提示词优化工具（拥有 2.1 万+ Stars），支持多端部署、多模型集成及高级测试功能，旨在帮助用户编写和迭代高质量的 AI 提示词。

**2) 核心要点**
*   **项目热度与状态**：项目主要使用 TypeScript 开发，当前最新版本为 v2.5.4，在 GitHub 上拥有 21699 Stars 和 2624 Forks。
*   **核心优化功能**：支持一键优化和多轮迭代，提供系统提示词与用户提示词双模式优化，并支持原始与优化后提示词的实时对比测试。
*   **多模型集成**：文本方面支持 OpenAI、Gemini、DeepSeek、智谱AI、SiliconFlow 等主流模型；图像方面支持文生图（T2I）和图生图（I2I），集成 Gemini、Seedream 等模型。
*   **高级测试模式**：提供上下文变量管理（自定义、批量替换）、多轮会话模拟测试以及工具调用（Function Calling）支持。
*   **安全与隐私架构**：采用纯客户端处理架构，数据直接与 AI 服务商交互，不经过中间服务器；同时支持配置访问密码以保障部署安全。
*   **多端支持与部署**：提供 Web 在线应用、桌面应用（可解决跨域 CORS 限制并支持自动更新）、Chrome 插件以及 Docker / Vercel 部署方式。
*   **MCP 协议支持**：支持 Model Context Protocol (MCP) 协议，可与 Claude Desktop 等兼容应用进行集成。
