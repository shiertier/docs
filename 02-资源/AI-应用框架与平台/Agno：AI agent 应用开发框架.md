## 摘要
**1) 一句话总结**
Agno 是一个基于 Python 的开源 AI 智能体应用框架，旨在帮助开发者大规模构建、运行和管理具备推理、工具调用、上下文记忆和决策能力的多智能体系统。

**2) 关键要点**
* **项目热度与版本**：项目在 GitHub 上拥有超 3.8 万 Stars 和 5000+ Forks，采用 Apache 2.0 开源协议，最新版本为 v2.5.3。
* **核心定位**：作为“智能体软件的编程语言”，推动软件从确定性的请求-响应模式向动态推理系统转变。
* **三层架构设计**：
  * **SDK 层**：负责智能体、团队、工作流、记忆、知识库、工具、护栏和审批流。
  * **Engine 层**：负责模型调用、工具编排、结构化输出和运行时强制执行。
  * **AgentOS 层**：提供流式 API、环境隔离、身份验证、审批执行、链路追踪和控制平面。
* **极简开发与部署**：仅需约 20 行代码即可构建一个带状态、可使用工具的智能体，并自动生成具备用户/会话隔离和原生追踪功能的生产级 API（默认运行于 localhost:8000）。
* **可视化管理**：开发者可通过连接 AgentOS UI (os.agno.com) 来监控、管理和测试本地或线上的智能体。
* **支持复杂交互模式**：将流式传输和长时运行作为一等公民，支持智能体实时输出推理过程，并允许中途暂停、等待审批及后续恢复。
* **内置安全与治理模型**：针对智能体动态决策的特点，提供审批工作流、人类介入（Human-in-the-loop）、审计日志及运行时权限强制执行功能。
