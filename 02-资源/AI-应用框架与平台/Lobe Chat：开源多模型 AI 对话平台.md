## 摘要
**1) 一句话总结**
Lobe-Chat 是一个基于 TypeScript 开发的现代 AI 应用框架与平台，致力于打造支持多模型、多智能体协作及人机共进的下一代工作与生活空间。

**2) 关键要点**
*   **项目热度与技术栈**：项目主要使用 TypeScript 开发，在 GitHub 上拥有超过 7.2 万 Stars 和 1.4 万 Forks，社区活跃度极高。
*   **核心理念**：以智能体（Agent）作为工作交互的基础单元，支持多智能体协作（Multi-agent collaboration）与团队设计，旨在构建人机共进（Co-evolution）网络。
*   **多模型支持**：兼容多种主流云端大语言模型（如 OpenAI, Claude, DeepSeek, Gemini 等），同时支持本地大语言模型（Local LLM）的接入。
*   **多模态能力**：内置模型视觉识别、TTS & STT（语音转文本/文本转语音）对话以及文本生成图片功能。
*   **高级对话特性**：支持智能互联网搜索、思维链（Chain of Thought）、分支对话、Artifacts 以及文件上传与知识库构建。
*   **强大的扩展性**：具备完善的插件系统（支持 Function Calling），提供 MCP（Model Context Protocol）插件一键安装与 MCP 市场，并内置智能体市场（GPTs）。
*   **数据与用户管理**：支持本地或远程数据库存储，具备多用户管理功能。
*   **跨平台与部署**：提供桌面端应用、PWA（渐进式 Web 应用）及移动端适配；支持通过 Vercel、Docker、Zeabur、阿里云等多种方式进行一键自托管部署。
