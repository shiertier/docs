## 摘要
### 一句话总结
Postiz 是一个基于 TypeScript 开发的开源 AI 社交媒体排期与管理工具，支持多平台发布、团队协作及数据分析，并提供面向 SaaS 企业的企业级云服务与丰富的 API 集成。

### 关键点
*   **项目热度与开源协议**：项目采用 AGPL-3.0 协议开源，目前拥有超过 2.6 万 Stars 和 4600+ Forks，最新版本为 v2.19.0。
*   **多平台支持**：支持集成 14 个主流社交与社区平台，包括 X、Instagram、LinkedIn、TikTok、Reddit、Discord、Threads 和 Mastodon 等。
*   **核心功能**：提供 AI 辅助的帖子排期、数据分析、团队协作（如评论、排期、帖子交换与购买）等功能，且自托管版本与托管版本功能一致。
*   **企业级云服务 (SaaS)**：推出支持多租户架构的企业版，提供 Headless API 和完整的 OAuth 支持，允许企业客户跳过繁琐的平台审批，直接在自有产品中集成社交媒体管理功能。
*   **自动化与开发者生态**：提供 Public API、NodeJS SDK，并原生支持 N8N（自定义节点）、Make.com 和 Zapier 等自动化平台；近期还推出了适用于 OpenClaw 等 Agent 的 Postiz agent CLI。
*   **技术栈**：采用 Pnpm Monorepo 架构，核心技术包括 NextJS (React)、NestJS、Prisma (默认 PostgreSQL)、Temporal 和 Resend。
*   **数据隐私与合规**：严格遵守平台合规性，使用官方批准的 OAuth 流程，不进行内容抓取，不收集或代理用户的 API 密钥，确保用户直接与社交平台进行身份验证。
