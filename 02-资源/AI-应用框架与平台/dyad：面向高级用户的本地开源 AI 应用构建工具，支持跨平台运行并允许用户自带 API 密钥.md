## 摘要
**一句话总结**
Dyad 是一个面向高级用户的本地开源 AI 应用构建工具，支持跨平台运行并允许用户自带 API 密钥，是 v0、Lovable 和 Bolt 等工具的本地替代方案。

**关键要点**
* **项目定位**：本地运行的 AI 应用构建器（AI-应用框架与平台），主打快速、隐私保护且无供应商锁定。
* **核心特性**：支持跨平台（Mac 和 Windows），用户可使用自己的 AI API 密钥（Bring your own keys），且无需注册即可下载使用。
* **项目数据**：主要使用 TypeScript 开发，在 GitHub 上拥有 19,666 个 Stars、2,231 个 Forks 和 109 名 Watchers。
* **版本动态**：项目创建于 2025年4月11日，最新版本为 v0.37.0-beta.2（发布于 2026年2月17日），近期持续更新。
* **开源协议**：采用混合许可模式，`src/pro` 目录外的代码基于 Apache 2.0 协议开源，`src/pro` 目录内的代码基于 Functional Source License 1.1 Apache 2.0 协议（Fair-source）。
* **社区生态**：在 Reddit 上设有官方社区（r/dyadbuilders），供开发者分享项目和获取帮助。
