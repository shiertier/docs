## 摘要
### 一句话总结
PDFMathTranslate 是一款基于 AI 的 PDF 科学论文双语翻译工具，能够在完整保留公式、图表等原有排版的基础上，提供多语言、多翻译引擎支持及多种部署方式。

### 核心要点
*   **核心功能**：实现 PDF 文档的全文双语翻译，严格保留原文档的排版布局，包括数学公式、图表、目录和注释。
*   **多翻译引擎支持**：兼容 Google、DeepL、Ollama、OpenAI 等多种主流 AI 翻译服务。
*   **丰富的使用形态**：提供命令行工具 (CLI)、图形用户界面 (GUI)、MCP、Docker 镜像以及 Zotero 插件，满足不同场景需求。
*   **免安装在线服务**：提供官方免费在线翻译网站 (pdf2zh.com)，并支持在 Immersive Translate (BabelDOC)、HuggingFace 和 ModelScope 平台上直接体验。
*   **本地安装便捷**：支持在 Python 环境（版本 3.10 至 3.12）下，通过 `uv` 或 `pip` 包管理工具快速进行本地安装与运行。
*   **版本与架构演进**：项目已推出 2.0 版本，并迁移至新组织仓库（`PDFMathTranslate/PDFMathTranslate-next`），近期更新增加了对 BabelDOC WebUI 的实验性支持以及打包完善的 Windows 可执行文件。
*   **开源与社区影响力**：项目主要使用 Python 开发，采用 GNU AGPL v3.0 开源协议，在 GitHub 上已获得超过 31,800 颗 Stars，社区关注度极高。
