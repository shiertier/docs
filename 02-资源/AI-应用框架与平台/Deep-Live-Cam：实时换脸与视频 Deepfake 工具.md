## 摘要
**一句话总结**
Deep-Live-Cam 是一个基于 Python 的开源实时换脸与视频 Deepfake 工具，提供实时替换与多人映射能力，同时在 README 中给出了伦理与合规约束声明。

**核心要点**
- 项目定位：面向实时 face swap 与视频 deepfake 处理场景。
- 主要能力：实时面部替换、多人映射、面向直播与视频流的处理能力。
- 合规提示：README 明确要求合法使用、获取被使用人同意，并在传播内容时标注 deepfake。
- 风险属性：该类工具具有明显滥用风险，涉及身份冒用、误导传播、隐私与合规问题。
- 许可信息：仓库许可为 GNU AGPL-3.0。

## 功能与定位
Deep-Live-Cam 的核心定位是提供实时人脸替换与视频 deepfake 处理能力。项目描述其面向 AI 生成媒体相关场景，强调“实时处理”与“低门槛体验”。

## 典型使用场景
- 实时视频中的人脸替换演示。
- 视频内容中的人物面部映射处理。
- 多人场景下的面部替换效果展示。

## 特色与差异点
- 聚焦实时处理链路，而非离线单次生成。
- 在项目说明中主动加入伦理和合法使用声明。
