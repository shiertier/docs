## 摘要

### 一句话总结
Bruno 是一款开源、轻量级且仅支持离线的 API 探索与测试 IDE，旨在作为 Postman 等工具的替代方案，支持通过本地文件和 Git 进行团队协作。

### 关键点
* **项目定位**：开源的 API 探索和测试 IDE，是 Postman 和 Insomnia 的轻量级替代品。
* **数据存储与格式**：API 集合直接存储在本地文件系统的文件夹中，使用纯文本标记语言（Bru）来保存请求信息。
* **版本控制协作**：支持通过 Git 或任何其他版本控制系统来管理和协作 API 集合。
* **隐私与离线优先**：完全离线运行，官方明确表示永远不会添加云同步功能，以确保用户数据隐私保留在本地设备上。
* **跨平台与安装**：支持 Mac、Windows 和 Linux 平台，并可通过 Homebrew、Chocolatey、Scoop、Apt 等多种主流包管理器进行安装。
* **商业模式**：绝大多数功能免费且开源（基于 MIT 协议），同时提供包含额外功能的付费商业版本，以平衡开源原则与项目的可持续性。
* **项目活跃度**：主要使用 JavaScript 开发，目前在 GitHub 上拥有超过 41,000 颗 Stars 和 2,100+ Forks，最新发布版本为 v3.1.3。
