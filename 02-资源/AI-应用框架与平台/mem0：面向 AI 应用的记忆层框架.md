## 摘要
**1) 一句话总结**
Mem0 是一个为 AI 智能体（AI Agents）提供通用智能记忆层的开源项目，旨在通过多级记忆和个性化适应能力，实现更高效、低成本的个性化 AI 交互。

**2) 关键要点**
*   **项目定位与功能**：为 AI 助手和智能体提供记忆层，能够记住用户偏好、适应个人需求并随时间持续学习，适用于 AI 助手、客户支持、医疗保健及游戏等场景。
*   **核心能力**：支持多级记忆（无缝保留用户、会话和智能体状态），提供开发者友好的 API、跨平台 SDK 以及全托管服务选项。
*   **性能优势**：根据官方研究（LOCOMO 基准测试），Mem0 的准确率比 OpenAI Memory 高 26%，响应速度比全上下文快 91%，Token 消耗降低 90%。
*   **部署与安装**：提供托管平台（Mem0 Platform）和开源自托管两种模式，开发者可通过 `pip install mem0ai` 或 `npm install mem0ai` 快速安装。
*   **模型支持**：运行需要依赖 LLM，默认使用 OpenAI 的 `gpt-4.1-nano-2025-04-14` 模型，同时也支持多种其他主流 LLM。
*   **最新版本更新**：最新大版本（v1.0.0 及以上）引入了 API 现代化、改进的向量存储支持以及增强的 GCP（谷歌云）集成。
*   **开源与社区状态**：项目采用 Apache License 2.0 协议，主要使用 Python 开发，社区热度极高（拥有超 4.7 万 Stars 和 5300+ Forks）。
