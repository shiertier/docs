## 摘要

**一句话总结**
NarratoAI 是一款基于大语言模型的免费一站式 AI 影视解说与自动化剪辑工具，支持从文案撰写、配音、字幕到视频生成的全链路工作流。

**关键要点**
* **项目数据**：主要使用 Python 开发，目前在 GitHub 拥有 8108 Stars 和 1058 Forks。
* **核心功能**：实现端到端的视频自动化处理，专设短剧处理专区，支持短剧解说、混剪、一键合并素材与转录。
* **AI 视频理解**：接入阿里 Qwen2-VL 大模型，具备直接理解视频内容的能力。
* **语音与配音**：支持多种 TTS 引擎（含腾讯云 TTS），并支持基于 IndexTTS2 的语音克隆功能。
* **多模型兼容**：通过 LiteLLM 管理模型供应商，支持接入 DeepSeek R1、DeepSeek V3 等最新大模型。
* **低硬件门槛**：运行该工具不强制要求配备独立显卡。
* **部署方式**：提供 macOS（Docker Compose 一键部署）、Windows（免配置整合包）以及本地源码（Streamlit Web UI）等多种运行方式。
* **项目背景**：基于开源项目 MoneyPrinter 和 MoneyPrinterTurbo 重构，专门扩展了影视解说功能。

## 功能与定位

NarratoAI 是一款完全免费的自动化影视解说与视频剪辑工具。该项目基于大语言模型（LLM），将文案撰写、视频剪辑、配音及字幕生成整合为一站式工作流，旨在帮助创作者高效完成影视及短剧内容的二次创作。

## 核心功能

- **全链路自动化**：支持从文案生成、配音、加字幕到视频剪辑的端到端自动化处理。
- **短剧处理专区**：
  - 支持短剧解说与短剧混剪。
  - 提供一键合并素材、一键转录、一键清理缓存等快捷功能。
- **AI 视频理解**：接入阿里 Qwen2-VL 大模型，具备直接理解视频内容的能力。
- **语音与配音**：
  - 支持多种 TTS（文本转语音）引擎，包括腾讯云 TTS。
  - 支持基于 IndexTTS2 的语音克隆功能。

## 特色与差异点

- **多模型兼容**：通过 LiteLLM 管理模型供应商，支持接入最新大模型（如 DeepSeek R1、DeepSeek V3 等）。
- **低硬件门槛**：运行该工具不强制要求配备独立显卡。
- **项目渊源**：基于开源项目 MoneyPrinter 和 MoneyPrinterTurbo 重构，并在其基础上专门扩展了影视解说功能。

## 使用方式概览

项目提供多种部署与运行方式，适配不同操作系统：
- **macOS 部署（推荐）**：通过 Docker Compose 一键部署运行。
- **Windows 部署（推荐）**：提供免配置的 Windows 整合包（需通过官方微信公众号获取下载链接）。
- **本地源码运行**：克隆仓库后安装 Python 依赖，配置 API 密钥，通过 Streamlit 启动 Web UI。
