## 摘要
**1) 一句话总结**
Kortix 是一个基于 TypeScript 开发的 AI 智能体平台，提供完整的工具链与基础设施，用于构建、管理和训练能够自主执行复杂任务的 AI 智能体。

**2) 关键要点**
*   **项目热度与技术栈**：该项目在 GitHub 上拥有 19,388 个 Stars 和 3,362 个 Forks，主要使用 TypeScript 开发，最新版本为 v1。
*   **核心定位**：作为一个全面的 AI 智能体平台，支持为各种用例（如客服、内容创作、销售营销、研发等）创建定制化的自主智能体。
*   **旗舰智能体**：内置通用型 AI 员工 “Kortix Super Worker”，可通过自然语言对话处理网络研究、数据分析、浏览器自动化和文件管理等复杂工作流。
*   **可视化构建工具**：提供 Agent Builder，用户可以通过可视化工具配置、定制和部署专属的 AI 智能体。
*   **网络与数据能力**：具备浏览器自动化（网页导航、数据提取、表单填写）和网络智能（爬虫、搜索、数据合成）功能。
*   **系统与文件操作**：支持文件管理（创建/编辑文档、表格、代码等）以及系统操作（安全执行命令行、系统管理和 DevOps 任务）。
*   **扩展与集成**：支持 API 集成，能够连接外部服务以实现跨平台的自动化工作流，并支持用户自托管（Self-Hosting）部署。
