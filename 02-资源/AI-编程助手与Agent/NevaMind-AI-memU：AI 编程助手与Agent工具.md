## 摘要

**一句话总结**
memU 是一款专为全天候（24/7）运行的主动型 Agent 提供持续记忆支持的 AI 编程助手与管理工具。

**关键要点**
* **核心定位**：作为 AI 编程助手与 Agent 工具，核心在于提供记忆（Memory）支持。
* **主要功能**：提供 Agent 记忆管理，赋予 AI 代理持续的记忆能力以支撑全天候运行。
* **设计特色**：专门针对 24/7 全天候在线的主动型（proactive）Agent 架构进行设计。
* **适用场景**：适用于需要长期维持状态并进行主动交互的 Agent 系统。
* **官方示例**：明确适用于 openclaw 及其相关机器人项目（如 moltbot、clawdbot）。
* **项目来源**：开源托管于 GitHub（NevaMind-AI/memU）。
