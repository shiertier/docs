## 摘要

**一句话总结**
MemOS 是一款专为大语言模型（LLM）和智能体（Agent）设计的 AI 记忆操作系统，通过提供持久化技能记忆，使 AI 能够打破单次任务局限，实现跨任务的技能复用与持续演进。

**核心要点**
* **系统定位**：专为 LLM 和 Agent 设计的 AI 记忆操作系统（AI memory OS），同时作为 AI 编程助手与 Agent 工具。
* **核心功能**：为 AI 系统提供持久化的技能记忆（Persistent Skill Memory）支持。
* **核心优势**：打破单次任务的局限，支持 AI 技能的跨任务复用与持续演进。
* **适用场景**：适用于需要在不同任务之间复用和进化 AI 技能的复杂场景。
* **生态集成**：可广泛集成于各类 LLM 和 Agent 系统中（如 moltbot、clawdbot、openclaw 等）。
* **项目数据**：截至 2026 年 2 月 14 日，其 GitHub 仓库已获得 5732 个 Stars。
