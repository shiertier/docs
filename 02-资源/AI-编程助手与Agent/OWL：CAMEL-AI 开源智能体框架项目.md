## 摘要
**1) 一句话总结**
OWL 是一个基于 CAMEL-AI 框架构建的多智能体协作框架，旨在通过动态智能体交互实现现实世界任务的高效自动化。

**2) 核心要点**
* **卓越性能**：在 GAIA 基准测试中获得 69.09 的平均分，在开源框架中排名第一。
* **技术基础**：建立在 CAMEL-AI 框架之上，专注于多智能体系统、Web 交互和任务自动化。
* **社区热度**：项目主要使用 Python 开发，拥有极高的社区关注度（19075 Stars，2224 Forks）。
* **部署方式**：提供多种安装选项，包括 uv（官方推荐）、venv/pip、conda 以及 Docker（支持预构建镜像或本地构建）。
* **工具与能力**：支持模型上下文协议（MCP）和 Playwright MCP 服务，并提供多模态工具包（需模型支持）和基于文本的工具包。
* **用户界面**：内置 Web UI 界面，方便用户启动和进行可视化交互。
* **学术支持**：项目相关的研究论文已发布（arXiv:2505.23885）。
