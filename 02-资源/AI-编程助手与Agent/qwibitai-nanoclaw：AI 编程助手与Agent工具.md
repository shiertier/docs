## 摘要

**1) 一句话总结**
nanoclaw 是一个基于 Anthropic Agents SDK 的轻量级 AI 编程助手与 Agent 工具，作为 Clawdbot 和 OpenClaw 的替代方案，提供 WhatsApp 接入、记忆、定时任务等功能，并通过容器化运行保障安全。

**2) 关键点**
* **项目定位**：一款轻量级的 AI 编程助手与 Agent 工具，是 Clawdbot 和 OpenClaw 的轻量化替代方案。
* **底层框架**：直接基于 Anthropic 的 Agents SDK 运行。
* **平台接入**：支持连接并接入 WhatsApp 平台。
* **记忆能力**：系统具备记忆（Memory）功能。
* **任务调度**：支持配置和执行定时任务（Scheduled jobs）。
* **安全机制**：通过在容器（Containers）中运行来保障系统的安全性。
* **开源地址**：项目托管于 GitHub (qwibitai/nanoclaw)。

## 功能与定位
nanoclaw 是一个 AI 编程助手与 Agent 工具，定位为 Clawdbot 和 OpenClaw 的轻量级替代方案。该项目直接基于 Anthropic 的 Agents SDK 运行。

## 核心功能
* **平台接入**：支持连接至 WhatsApp。
* **记忆能力**：具备记忆（Memory）功能。
* **任务调度**：支持执行定时任务（Scheduled jobs）。
* **底层框架**：直接运行在 Anthropic 的 Agents SDK 之上。

## 特色与差异点
* **轻量化**：相比 Clawdbot 和 OpenClaw，提供更轻量级的解决方案。
* **安全性**：通过在容器（Containers）中运行来保障系统安全。
