# Factory IDE：终端与编辑器中的 AI coding agent

## 文档信息

- 对象：网站产品页
- 来源：https://factory.ai/product/ide
- 来源：https://factory.ai/pricing
- 来源：https://factory.ai/enterprise

## 摘要

Factory IDE 是一款集成于终端和主流编辑器中的 AI 编程智能体，旨在通过多模型支持、超大代码库检索和企业级安全管控，协助开发者完成从代码生成、测试、审查到合并的完整软件开发流程。

- 全流程开发辅助：覆盖 Generate、Test、Review、Document、Merge 等环节。
- 平台兼容性：支持 VS Code、JetBrains、Vim 与终端，并提供浏览器、CLI 等多入口工作流。
- 多模型选择：官方文案显示可按任务选择 Claude、GPT、Gemini 等模型系列。
- 大代码库能力：强调面向大规模代码库的 agentic search 与上下文理解。
- 企业能力：提供 SSO/SAML/SCIM、审计日志、合规报告、专属算力与本地部署选项。
- 官方指标声明：页面展示了 7x faster、96.1% reduction、95.8% time saved 等效果数据。
- 定价层级：提供 Pro（$20/月）、Max（$200/月）和 Enterprise 三档方案。

- 套餐边界：Pro/Max 存在 Token 与席位上限。
- 企业功能门槛：无限成员、本地部署及部分高级管理能力仅在 Enterprise 提供。

## 功能与定位

Factory IDE 将自身定位为可在开发者现有工具链中运行的 coding agent，不要求开发者切换 IDE。产品强调“智能体在你写代码的地方工作”，并把能力扩展到从编码到交付的完整流程。

## 典型使用场景

- 在 IDE 或终端中协作式生成、修改与审查代码。
- 在大型仓库中做跨文件定位与改动编排。
- 在企业研发流程中结合权限审批与审计能力进行 AI 协作开发。
- 结合 CLI 与外部协作入口做任务委派和自动化。

## 核心功能

- 多端开发入口：IDE + 终端并行支持，覆盖 VS Code、JetBrains、Vim 等环境。
- 终端能力集：支持斜杠命令、自定义 subagents、原生 diff 视图、上下文标记。
- 自主性控制：支持可调自治级别与人工审批边界。
- 模型可切换：按任务选择模型，避免单模型绑定。
- 企业能力：SSO/SAML/SCIM、审计日志、合规报告、专属算力、本地部署。

## 特色与差异点

- 工作流嵌入式设计：强调“不切 IDE”而不是新建独立操作界面。
- 明确的控制模型：强调先审阅再落盘，降低意外改动风险。
- 企业导向较强：定价与功能同时覆盖个人、小团队到企业场景。

## 使用方式概览

- 从产品页进入 CLI 安装入口（支持 macOS/Linux 与 Windows）。
- 按 Quickstart 指引完成初始化和首次任务。
- 按套餐选择协作规模与企业能力。

## 限制与注意事项

- 页面中的性能与 ROI 数据属于官方声明，使用前建议结合自身场景验证。
- Pro/Max 的 Token 与席位上限会影响团队规模与并发使用方式。
- 企业专属能力（如本地部署、无限成员）并非所有计划默认可用。
