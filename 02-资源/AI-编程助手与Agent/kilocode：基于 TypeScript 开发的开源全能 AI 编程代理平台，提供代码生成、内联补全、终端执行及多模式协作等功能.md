## 摘要
**1) 一句话总结**
Kilo (kilocode) 是一个基于 TypeScript 开发的开源全能 AI 编程代理平台，提供代码生成、内联补全、终端执行及多模式协作等功能，并通过 VS Code 插件支持调用 500 多种前沿 AI 模型。

**2) 关键要点**
* **项目定位与热度**：定位为全能型 AI 编程代理平台，在 OpenRouter 排名第一，拥有超 150 万用户，处理了超 25 万亿 tokens，GitHub Stars 超过 1.5 万。
* **核心编程功能**：支持通过自然语言生成代码、AI 驱动的内联代码补全、自动化重复性任务以及高效的现有代码重构。
* **高级代理能力**：Agent 能够自我检查工作、运行终端命令以及实现浏览器自动化操作。
* **多模式工作流（Multi Mode）**：内置架构师（Architect，用于规划）、程序员（Coder，用于编码）和调试员（Debugger，用于调试）模式，并支持用户创建自定义模式。
* **强大的扩展性**：内置 MCP 服务器市场（MCP Server Marketplace），用户可轻松查找并使用 MCP 服务器来扩展 Agent 的能力。
* **广泛的模型支持**：支持 500 多种最新 AI 模型（包括 Gemini 3 Pro、Claude 4.5 Sonnet & Opus、GPT-5），API 密钥为可选配置，平台定价与模型供应商官方费率完全一致。
* **使用与奖励机制**：主要通过 VS Code 扩展市场下载使用，新用户首次充值可获得 20 美元的奖励积分。
* **开源协议**：项目主要使用 TypeScript 编写，采用宽松的 Apache License 2.0 协议，允许包含商业用途在内的使用、修改和分发。
