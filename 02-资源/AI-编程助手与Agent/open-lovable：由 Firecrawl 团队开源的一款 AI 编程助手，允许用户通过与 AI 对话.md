## 摘要
**一句话总结**
Open Lovable 是由 Firecrawl 团队开源的一款 AI 编程助手，允许用户通过与 AI 对话，在几秒钟内克隆并生成现代 React 应用程序。

**关键点**
*   **项目定位**：属于 AI 编程助手与 Agent 类别，旨在通过对话即时构建 React 应用，完整云端解决方案指向 Lovable.dev。
*   **项目热度**：社区关注度极高，目前已获得 24,217 个 Stars 和 4,718 个 Forks。
*   **技术栈**：主要使用 TypeScript 开发，本地运行默认端口为 `localhost:3000`。
*   **核心依赖**：项目运行强制要求配置 Firecrawl API 密钥（`FIRECRAWL_API_KEY`）。
*   **多模型支持**：兼容多种主流 AI 大模型，用户可自由选择配置 Gemini、Anthropic、OpenAI 或 Groq 的 API 密钥。
*   **沙盒环境**：支持两种沙盒提供商，默认为 Vercel（支持 OIDC Token 或个人访问令牌认证），也可切换为 E2B 沙盒。
*   **可选加速功能**：支持配置 Morph API 密钥（Fast Apply）以实现更快的代码编辑速度。
*   **开源协议**：项目遵循 MIT 开源许可协议。
