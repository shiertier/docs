## 摘要
**1) 一句话总结**
Legado（阅读3.0）是一款基于 Kotlin 开发的 Android 免费开源阅读器，允许用户通过高度自定义的规则抓取网络内容或导入本地文件，从而获得无广告的个性化阅读体验。

**2) 核心要点**
* **项目热度与规范**：项目采用 Kotlin 编写，遵循 GNU GPL v3.0 开源协议，在 GitHub 上拥有超 4.4 万 Stars 和近 5000 Forks。
* **内容获取机制**：软件自身不提供任何书籍内容，完全依赖用户手动添加自定义书源（通过规则抓取网页数据）或订阅 RSS 内容。
* **本地文件支持**：支持本地 TXT 和 EPUB 格式文件的智能扫描与手动浏览阅读。
* **阅读界面自定义**：提供高度可定制的阅读界面，支持切换字体、颜色、背景、行段间距、简繁体转换，以及覆盖、仿真、滑动、滚动等多种翻页模式。
* **内容净化功能**：内置替换净化功能，用户可设置规则轻松去除广告或替换文本内容。
* **API 与便捷导入**：提供 Web 和 Content Provider 两种 API 接口；支持通过 `legado://import/` URL 协议一键导入书源、订阅源、替换规则、主题和阅读排版等配置。
* **周边生态与社区**：项目配套有 Web 端书架和 Web 端源编辑器，并拥有完善的官方帮助文档以及活跃的 Telegram、Discord 交流社区。
