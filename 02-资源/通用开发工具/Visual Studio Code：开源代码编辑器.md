## 摘要
**1) 一句话总结**
Visual Studio Code (VS Code) 是由微软与社区共同开发的开源代码编辑器（"Code - OSS"），提供全面的代码编辑、轻量级调试及丰富的扩展模型。

**2) 关键要点**
* **项目热度与技术栈**：主要使用 TypeScript 开发，在 GitHub 上拥有超 18.1 万 Stars 和 3.8 万 Forks，开源代码基于 MIT 协议。
* **开源与商业版区别**：官方下载的 VS Code 产品是基于此开源仓库（"Code - OSS"）并加入微软特定定制后，以传统微软产品许可证发布的版本。
* **核心功能**：结合了代码编辑器的简洁性与开发者所需的“编辑-构建-调试”核心循环，支持代码导航、理解、轻量级调试及工具集成。
* **更新与发布机制**：项目每月进行更新（包含新功能和 Bug 修复），并提供每日更新的 Insiders 预览版；最新发布版本为 1.109.5。
* **高度透明的规划**：除了代码和 Issue，该仓库还公开了项目的路线图（roadmap）、每月迭代计划（iteration plans）和发布冲刺计划（endgame plans）。
* **模块化与扩展**：许多核心组件和扩展（如 Node/Mono 调试适配器）在独立的 GitHub 仓库中维护；仓库内的 `extensions` 文件夹包含了内置的语法、代码片段及提供丰富语言支持的扩展。
* **开发环境要求**：原生支持 Dev Containers 和 GitHub Codespaces，运行完整构建的容器环境至少需要 4 核 CPU 和 6 GB 内存（推荐 8 GB）。
