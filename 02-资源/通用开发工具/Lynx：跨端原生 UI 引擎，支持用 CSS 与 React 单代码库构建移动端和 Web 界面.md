## 摘要

### 1) 一句话总结
Lynx 是一个基于 C++ 开发的开源跨平台核心引擎，允许开发者使用现有的 Web 技能（如 CSS 和 React）通过单一代码库构建高性能的移动端和 Web 端原生 UI。

### 2) 关键点
*   **项目定位**：作为 Lynx 家族的核心引擎，支持“一次编写，随处渲染”，提供 Android、iOS 和 Web 的原生渲染，或通过自定义渲染器实现移动端和桌面端的像素级一致性。
*   **技术与性能**：主要使用 C++ 开发，采用多线程引擎以实现应用的瞬间启动和流畅的 UI 响应。
*   **运行要求**：目标应用需运行在 iOS 10 及 Android 5.0 (API 21) 或更新版本的系统上。
*   **开发环境**：官方推荐使用 macOS 作为开发操作系统。
*   **社区与热度**：项目采用 Apache License 2.0 开源协议，社区关注度极高（超 14,400 Stars，500 Forks），最新发布版本为 3.6.0。
*   **技术生态**：集成了 V8、BoringSSL、GoogleTest 等多个第三方开源库。
*   **架构借鉴**：其 API 设计与底层实现参考了 Chromium、React Native 和 Flutter Engine 等业界知名项目。

### 3) 风险/缺口
*   **开发环境限制**：官方明确表示 Windows 和 Linux 操作系统尚未经过验证或提供保证，开发者在这些系统上可能会遇到问题。
