## 摘要
**一句话总结**
community-scripts/ProxmoxVE 是一个由社区接管维护的开源 Shell 脚本库，旨在通过一键式自动化脚本极大地简化 Proxmox VE 的环境配置与服务安装。

**关键要点**
* **项目背景**：该项目为纪念原作者 tteck 而由社区继续维护，采用 MIT 开源协议，目前在 GitHub 上拥有超过 2.6 万 Stars。
* **核心功能**：提供常用服务和容器的一键安装、灵活配置（支持新手与高级模式）、自动更新机制以及安装后的管理与故障排除功能。
* **环境要求**：需要 Proxmox VE 8.4.x、9.0.x 或 9.1.x 版本，基于 Debian 的操作系统（含 Proxmox Tools），并要求设备联网。
* **Web 一键安装**：用户可在官方网站 (helper-scripts.com) 搜索所需脚本，复制 Bash 命令并在 Proxmox Shell 中直接运行即可完成安装。
* **本地 UI 管理**：支持通过执行特定命令安装 `PVEScripts-Local`，从而在 Proxmox 界面中直接添加脚本管理菜单，无需频繁访问网站。
* **社区与生态**：项目拥有活跃的 Discord 社区和 GitHub 讨论区，鼓励用户参与代码编写、文档完善和脚本测试。
* **公益捐赠**：项目接受赞助以维护基础设施，且明确承诺将 30% 的捐款直接用于癌症研究和临终关怀机构。
