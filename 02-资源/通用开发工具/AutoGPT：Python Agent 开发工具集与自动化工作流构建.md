## 摘要
**1) 一句话总结**
AutoGPT 是一个高人气的 Python 通用开发平台，旨在让用户能够通过低代码界面、标准化协议和丰富的工具链，轻松构建、部署和管理用于自动化复杂工作流的持续性 AI 代理。

**2) 关键要点**
* **项目热度与状态**：该项目在 GitHub 上拥有超 18.1 万 Stars 和 4.6 万 Forks，最新测试版本为 `autogpt-platform-beta-v0.6.48`。
* **部署选项**：支持免费的本地自托管（提供适用于各操作系统的一键安装脚本），并提供云端托管服务（目前处于封闭测试排队阶段）。
* **硬件与软件要求**：本地部署推荐配置为 4核+ CPU、16GB 内存及 10GB 存储空间；依赖 Docker、Docker Compose、Git、Node.js 和 npm 等基础软件。
* **前端架构 (Frontend)**：提供低代码代理构建器、基于区块连接的工作流管理、部署控制、性能监控分析，以及开箱即用的预置代理库。
* **后端服务 (Server)**：作为平台的核心引擎，负责持续运行代理、提供底层基础设施支持，并集成了代理应用市场。
* **双重开源协议**：`autogpt_platform` 目录下的代码采用 Polyform Shield 协议；项目其余部分（包括经典版 AutoGPT、Forge、Benchmark 等）均采用 MIT 协议。
* **经典版工具链 (Classic)**：包含用于快速开发代理的 Forge 工具包、用于客观评估代理性能的 `agbenchmark`，以及统一的命令行工具（CLI）。
* **标准化通信**：采用 AI Engineer Foundation 发布的 Agent Protocol 标准，确保代理与前端及基准测试工具之间的通信兼容性。
