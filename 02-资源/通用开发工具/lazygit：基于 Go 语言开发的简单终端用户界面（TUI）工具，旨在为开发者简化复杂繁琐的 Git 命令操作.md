## 摘要
**1) 一句话总结**
Lazygit 是一个基于 Go 语言开发的简单终端用户界面（TUI）工具，旨在为开发者简化复杂繁琐的 Git 命令操作。

**2) 关键要点**
*   **项目热度与规范**：该项目在 GitHub 上拥有超过 72,800 个 Stars 和 2,500+ 个 Forks，采用 MIT 开源协议，主要使用 Go 语言开发。
*   **核心痛点解决**：专为解决原生 Git 操作繁琐的问题而设计，免去了手动编辑 TODO 文件或复杂补丁文件的麻烦，使交互式变基、部分代码暂存等高级操作更加直观。
*   **丰富的功能集**：支持逐行暂存（Stage individual lines）、交互式变基、Cherry-pick、二分查找（Bisect）、撤销/重做（Undo/Redo）、查看提交图谱以及对比提交等核心 Git 功能。
*   **高级与定制化支持**：支持多工作区（Worktrees）、自定义命令、自定义分页器（Pagers）以及 Git flow 工作流。
*   **广泛的跨平台安装**：支持 macOS、Linux 和 Windows，并可通过 Homebrew、Scoop、Winget、Chocolatey、APT 等数十种主流包管理器进行安装。
*   **项目活跃度**：项目持续维护更新，当前最新发布版本为 v0.59.0。
