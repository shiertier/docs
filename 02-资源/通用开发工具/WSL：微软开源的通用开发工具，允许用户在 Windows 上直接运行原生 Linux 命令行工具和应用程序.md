## 摘要
**一句话总结**
Windows Subsystem for Linux (WSL) 是微软开源的通用开发工具，允许用户在 Windows 上直接运行原生 Linux 命令行工具和应用程序，而无需传统虚拟机或双系统的开销。

**关键点**
* **核心功能**：在 Windows 环境中直接运行未经修改的 Linux 命令行工具、实用程序和应用程序。
* **项目数据**：主要使用 C++ 开发，采用 MIT 开源协议，在 GitHub 上拥有超过 3.1 万 Stars 和 1600+ Forks。
* **快速安装**：用户只需在 Windows 命令行中执行 `wsl --install` 即可完成安装。
* **生态系统**：包含多个相关的开源子项目，如 `WSL2-Linux-Kernel`（WSL 搭载的 Linux 内核）、`WSLg`（Linux GUI 应用支持）以及 `wsl`（官方文档）。
* **社区参与**：欢迎代码、文档修复和设计提案等贡献，并提供了包含源码构建说明和架构细节的开发者指南。
* **数据与隐私**：软件默认会收集基础诊断数据（遥测）以改进产品，用户可按说明关闭该功能；若开发者使用相关功能收集用户数据，必须遵守适用法律及微软隐私声明。
* **版本动态**：项目持续更新，最新发布版本为 2.6.3（发布于 2025 年 12 月 12 日）。
