## 摘要
**一句话总结**
Graphite 是一款基于 Rust 开发的开源 2D 内容创作工具套件，结合了图层合成与基于节点的程序化生成设计，提供完全非破坏性的编辑工作流。

**关键要点**
* **核心定位**：一款免费、开源的矢量与光栅图形引擎，目前处于 Alpha 测试阶段。
* **技术栈与协议**：主要使用 Rust 语言开发，代码主要采用 Apache License 2.0 开源协议（部分目录采用 MIT/Apache 2.0 双重许可）。
* **项目数据**：在 GitHub 上拥有 24,303 颗 Stars 和 1,086 个 Forks，最新稳定版本发布于 2022 年 2 月。
* **核心特性**：采用类似游戏引擎的底层架构，以节点图（Node Graph）为核心，为用户提供友好的矢量和光栅处理工作流。
* **未来路线图**：项目正从单一的矢量编辑器向全能图形工具箱演进，计划支持照片编辑、动态图形、数字绘画、桌面出版和视觉特效（VFX）合成等功能。
* **运营模式**：项目 100% 由社区构建和资助，其愿景是通过持续创新成为行业标准的综合性艺术与设计工具。
