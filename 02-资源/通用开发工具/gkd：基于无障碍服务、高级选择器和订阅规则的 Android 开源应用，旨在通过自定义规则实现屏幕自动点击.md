## 摘要

**1) 一句话总结**
GKD 是一款基于无障碍服务、高级选择器和订阅规则的 Android 开源应用，旨在通过自定义规则实现屏幕自动点击，以简化重复操作或跳过繁琐流程。

**2) 关键要点**
* **基本信息**：项目采用 Kotlin 语言开发，遵循 GPL-3.0 开源协议，在 GitHub 上拥有超 3.6 万 Stars，最新发布版本为 v1.11.6。
* **核心功能**：通过自定义规则，在指定界面满足特定条件（如屏幕存在特定文字）时，自动点击目标节点、位置或执行其他操作。
* **技术实现**：结合高级选择器（类似 CSS 选择器，支持联系节点上下文信息）、订阅规则与快照审查技术，实现目标节点的精确查找与定位。
* **主要应用场景**：用于执行快捷操作（如自动确认电脑登录）以及跳过应用启动时的繁琐流程（如开机广告等）。
* **规则订阅机制**：应用默认不提供任何规则，用户必须自行添加本地规则或通过链接导入远程订阅规则。
* **生态与扩展**：官方提供 `subscription-template` 帮助用户快速构建专属远程订阅，用户也可在 GitHub 通过 `gkd-subscription` 标签查找第三方订阅列表。
* **下载渠道**：支持通过官方主页 (gkd.li)、Google Play 商店以及 GitHub Releases 进行下载安装。
* **合规与免责**：官方明确声明项目仅供学习交流使用，严禁用于商业或非法用途。
