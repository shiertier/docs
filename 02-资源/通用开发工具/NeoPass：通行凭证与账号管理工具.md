## 摘要
### 1) 一句话总结
NeoPass 是一款基于 JavaScript 的高人气 Chrome 浏览器扩展（拥有超 2.5 万 Stars），旨在绕过 Iamneo、HackerRank 和 NPTEL 等在线考试平台的防作弊检测，并提供 AI 辅助答题功能。

### 2) 核心亮点
*   **适用平台**：专为在 Chrome 浏览器中运行的 Iamneo portal、HackerRank、NPTEL（如野生动物生态学、保护地理学等）及其他受限考试平台设计。
*   **检测绕过功能**：能够伪装并绕过 Examly 的 NeoExamShield 限制，同时提供防止标签页切换检测（Tab Switching Bypass）和解除粘贴限制功能。
*   **AI 辅助答题**：内置隐身模式的 AI 聊天机器人，支持通过快捷键一键解答选择题（MCQ）、搜索代码答案，甚至支持“逐字自动输入”代码以模拟人工操作。
*   **免费版（自带 API）**：免费用户无需注册，但需在设置中自行配置 AI API Key，支持 OpenAI、Google Gemini、Anthropic Claude 及自定义端点。
*   **Pro 版专属特性**：付费版提供免配置的托管 AI（标称为 GPT-5.1）、内置考试助手的专属 NeoBrowser、更高的请求速率上限，并能无视网络屏蔽限制。
*   **快捷键驱动**：为 Windows/Linux 和 Mac 用户提供了一套完整的快捷键（如 `Alt+Shift+Q` 求解问题，`Ctrl+V` 强制粘贴等），部分功能需先选中文本。
*   **安装方式**：未上架扩展商店，需下载源码压缩包并通过 Chrome 的“开发者模式（加载已解压的扩展程序）”进行手动安装。

### 3) 风险与局限（基于项目声明）
*   **学术诚信与违规风险**：项目明确声明“仅供教育目的使用”，不鼓励或促进作弊，并警告用户需谨慎使用以维护学术诚信，开发者对用户的任何行为不承担责任。
*   **网络屏蔽限制**：对于免费版用户，如果所在学校或组织在网络层屏蔽了 AI 服务提供商（如 OpenAI、Google 等），即使 API Key 有效扩展也无法工作，需依赖 VPN 或升级至 Pro 版。
