## 摘要
### 一句话总结
Better Auth 是一个基于 TypeScript 的、与框架无关的开源身份验证与授权框架，旨在通过开箱即用的功能和丰富的插件生态简化复杂的认证需求。

### 关键要点
*   **项目定位**：专为 TypeScript 打造的全面身份验证与授权库，且与具体框架无关（Framework-agnostic）。
*   **核心特性**：提供开箱即用的基础功能，并内置插件生态系统，开发者只需极少代码即可快速集成高级功能。
*   **高级功能支持**：通过插件可轻松实现双因素认证（2FA）、多租户（multi-tenant）支持等复杂业务场景。
*   **项目初衷**：旨在解决 TypeScript 生态中身份验证方案不完善的问题，替代需要大量样板代码的开源库或纯第三方服务。
*   **项目数据**：截至最新记录，该项目在 GitHub 拥有 26,384 个 Stars 和 2,278 个 Forks，最新发布版本为 v1.5.0-beta.16。
*   **技术标签**：主要语言为 TypeScript，涵盖 authentication、iam、oauth2、oidc、sso 等技术领域。
*   **开源与安全**：采用 MIT 开源协议；设有专门的安全漏洞报告邮箱（security@better-auth.com），并承诺及时处理漏洞报告。
