## 摘要
**1) 一句话总结**
`ui-ux-pro-max-skill` 是一个基于 Python 的开源 AI 技能库，通过内置的推理引擎和海量预设资源，为多平台和框架自动生成专业的 UI/UX 设计系统及代码实现。

**2) 关键点**
*   **核心功能**：v2.0 引入了“智能设计系统生成器”，可根据用户需求自动输出包含页面模式、UI 风格、配色、排版、动效、反模式（需避免的设计）及交付前检查清单的完整设计系统。
*   **工作原理**：通过 5 个维度的并行搜索（产品类型、风格、配色、页面模式、字体），结合基于 BM25 排序和 JSON 条件的推理引擎来生成最终设计。
*   **行业推理规则**：内置 100 条特定行业的推理规则，覆盖科技/SaaS、金融、医疗、电商、服务、创意及新兴技术等领域。
*   **海量设计资产**：包含 67 种 UI 风格（49 种通用、8 种落地页、10 种 BI/数据看板）、96 种配色方案、57 种字体搭配、25 种图表类型以及 99 条 UX 指南。
*   **多技术栈支持**：支持 13 种主流技术栈，包括 React, Next.js, Vue, Nuxt, Svelte, SwiftUI, Flutter, Tailwind 等。
*   **广泛的 AI 助手集成**：支持通过 CLI (`uipro-cli`) 或 Claude Marketplace 安装，可无缝集成至 Claude Code, Cursor, Windsurf, GitHub Copilot, Trae 等多种 AI 编程助手，提供自动激活（Skill Mode）或斜杠命令（Workflow Mode）两种使用方式。
*   **环境依赖**：核心搜索脚本的运行需要依赖 Python 3.x 环境。
*   **项目热度**：采用 MIT 开源协议，社区关注度极高（超 33,400 Stars，3,300+ Forks），当前最新版本为 v2.2.1。
