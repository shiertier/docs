## 摘要
**1) 一句话总结**
Maple Mono 是一款专为 IDE 和终端设计的开源圆角等宽编程字体，支持智能连字、Nerd-Font 图标、可变字体格式以及中英文 2:1 完美对齐。

**2) 关键要点**
* **项目热度与协议**：项目在 GitHub 上拥有超 2.3 万 Stars，主要使用 Python 开发，采用 SIL Open Font License 1.1 开源协议。
* **V7 版本重制**：V7 版本进行了全面重制，提供了可变字体格式和项目源文件，重绘了超过一半的字形，并提供了更智能的连字功能（最新版本为 v7.9）。
* **核心设计**：采用圆角设计，对 `@ $ % & Q ->` 等符号进行了全新设计，并在斜体样式中引入了连笔的 `f i j k l x y`。
* **图标与连字**：内置大量实用的智能连字功能，并提供对 Nerd-Font 的原生支持，增强终端的视觉体验。
* **多语言与排版**：中文版基于 Resource Han Rounded，支持简体中文、繁体中文和日语；实现了中英文宽度完美的 2:1 对齐，非常适合 Markdown 表格等多语言混合排版场景。
* **高度可定制**：提供细粒度的自定义选项，用户可以根据个人喜好自由启用或禁用特定的字体特性。
* **跨平台安装**：支持通过包管理器快速安装，包括 Windows 的 Scoop 以及 MacOS/Linux 的 Homebrew。

**3) 风险/不足**
* 官方文档明确指出，与其他流行的中文字体相比，该字体的中文汉字间距较大。

## 功能与定位
Maple Mono: Open source monospace font with round corner, ligatures and Nerd-Font icons for IDE and terminal, fine-grained customization options. 带连字和控制台图标的圆角等宽字体，中英文宽度完美2:1，细粒度的自定义选项
