## 摘要
### 一句话总结
Zen Browser 是一款基于 Firefox 构建的开源桌面浏览器，主要使用 JavaScript 开发，旨在提升用户的生产力并提供更宁静的上网体验。

### 关键要点
* **项目定位**：一款基于 Firefox 的浏览器，核心目标是将用户的生产力提升到新水平（"Welcome to a calmer internet"）。
* **底层依赖**：当前正式版（Release）基于 Firefox 147.0.4 构建，测试版（Twilight）基于 Firefox RC 148.0 构建。
* **技术与协议**：项目主要使用 JavaScript 语言开发，采用 Mozilla Public License 2.0 开源协议。
* **社区热度**：在 GitHub 上拥有极高的关注度，累计获得 40,237 Stars 和 1,339 个 Forks。
* **项目动态**：项目创建于 2024 年 3 月，最新版本为 1.18.10b（发布于 2026 年 2 月 21 日）。
* **社区协作**：官方建议通过 GitHub Issues 报告 Bug，通过 Github Discussions 提交功能请求，并提供详细的贡献指南欢迎社区参与。
* **合作伙伴**：项目官方列出了 Blacksmith 作为其支持与贡献的合作伙伴。
