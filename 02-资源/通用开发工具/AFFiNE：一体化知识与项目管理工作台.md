## 摘要
### 一句话总结
AFFiNE 是一款基于 TypeScript 开发的开源、本地优先且注重隐私的下一代知识库工具，将文档、白板和表格深度融合，旨在成为 Notion 和 Miro 的强大替代品。

### 核心要点
* **项目热度与技术栈**：该项目在 GitHub 上拥有超过 63,000 个 Stars 和 4,400 个 Forks，主要使用 TypeScript 开发。
* **无边界画布（Edgeless Canvas）**：打破传统文档限制，允许在同一画布上自由组合富文本、便签、嵌入网页、多视图数据库、图形和幻灯片等任意模块。
* **多模态 AI 助手**：内置 AFFiNE AI，支持撰写专业报告、将大纲转化为幻灯片、总结文章生成思维导图，甚至通过提示词直接绘制和编写原型应用代码。
* **本地优先与实时协作**：强调数据主权（数据保存在本地磁盘），同时支持 Web 端和跨平台客户端的实时同步与团队协作。
* **私有化部署**：支持用户通过 Docker 进行自托管（Self-host），完全掌控功能丰富的无限制版本。
* **底层核心技术**：项目由多个强大的开源引擎驱动，包括 BlockSuite（协同编辑器）、y-octo（高性能 YJS CRDT 实现）和 OctoBase（Rust 编写的轻量级本地优先数据库）。
* **开源协议与版本规划**：当前的社区版（CE）基于 MIT 协议免费开源；未来计划发布企业版（EE），将提供品牌重塑、SSO 单点登录及高级后台审计等企业级功能。
* **丰富的模板生态**：官方及社区提供了大量预置模板（如愿景板、教案、日程表、康奈尔笔记等），方便用户快速上手。
