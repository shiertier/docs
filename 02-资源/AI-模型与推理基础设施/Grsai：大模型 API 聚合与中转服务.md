## 摘要

**1) 一句话总结**
Grsai 是一个主打低价、高可用且直连官方源头的主流 AI 大模型 API 聚合平台，为开发者提供统一的文本、图像和视频生成接口调用服务。

**2) 核心要点**
* **多模型聚合**：统一提供 GPT-4o、Gemini、Claude、Flux、Veo、Sora 等主流文本、图像及视频生成模型的 API 接口。
* **源头直连**：由专人维护并直接对接官方接口，明确拒绝任何第三方平台转接，以保障服务稳定性与问题修复效率。
* **高性能与高可用**：采用多节点部署与全球加速线路，支持自动负载均衡与智能故障转移；官方承诺 99.99% 的可用性，平均响应时间小于 200ms。
* **低成本定价**：主打市场低价，例如 GPT Image 和 sora-image 单次请求低至 0.02 元。
* **统一计费管理**：采用积分制计费，不同模型消耗对应积分，平台提供多种充值套餐。
* **新用户福利**：新注册用户可获赠 5000 积分用于接口体验与测试。
* **技术支持**：提供 24/7 全天候实时在线技术支持（通过邮件联系）。

**3) 风险与不足**
* **计费差异大**：不同模型的调用成本（积分消耗）差异较大（例如 `sora-image` 为 0.02 元/次，而 `veo3.1-pro` 高达 0.4 元/次），开发者在使用前必须仔细查阅具体模型的价格表以控制成本。

## 功能与定位
Grsai 是一个聚合了多种主流 AI 大模型的 API 服务平台，提供文本生成、图像创作、视频生成等接口。该平台主打高并发、低延迟、低价格，并强调直接对接官方源头接口以保障稳定性。

## 典型使用场景
开发者或企业需要以低成本、高稳定性的方式，统一接入和调用多种主流 AI 模型（涵盖文本、图像、视频生成），而无需分别对接各个官方平台。

## 核心功能
* **多模型聚合**：支持 GPT-4o、Gemini、Claude、Flux 绘画、Veo 视频生成、Sora 等主流 AI 模型的接口调用。
* **高性能基础架构**：采用多节点部署与全球加速访问（覆盖国内外优质线路），支持自动负载均衡、峰值扩容、实时监控与智能故障转移。
* **统一计费管理**：采用积分制计费，不同模型消耗不同积分，支持充值套餐。

## 特色与差异点
* **源头直连**：专人维护 API，直接对接官方，明确拒绝转接任何第三方平台，以确保遇到问题可及时修复。
* **高可用与低延迟**：官方承诺提供 99.99% 的服务可用性保证，平均响应时间小于 200ms。
* **价格低廉**：主打市场低价，例如 GPT Image 模型、sora-image 等单次请求低至 0.02 元。
* **全天候支持**：提供 24/7 实时在线的技术支持（可通过邮件联系）。

## 使用方式概览
1. 注册平台账户（新注册用户可获赠 5000 积分用于体验）。
2. 在控制台创建 API 密钥。
3. 参考平台提供的详细文档和示例代码，将 API 接口集成到自身项目中。

## 限制与注意事项
* 平台采用积分制计费，不同模型的调用成本（积分消耗）差异较大（如 `sora-image` 为 0.02 元/次，而 `veo3.1-pro` 为 0.4 元/次），使用前需查阅具体模型的价格表。
