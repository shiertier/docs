## 定义

xAI 是 Elon Musk 创立的人工智能公司，官方使命是通过 AI 推进人类对世界的理解与能力边界。在本仓库里，xAI 通常对应 Grok 模型家族、grok.com 与 X 上的产品形态，以及面向开发者和团队的 xAI API / Enterprise 能力。

## 核心内涵

xAI 不只代表单一模型提供方，也代表一个同时覆盖聊天助手、开发者 API、企业交付与多模态生成能力的产品生态。按 2026-03 核对的 xAI 官方站点，其对外入口已明确分为 Grok 应用、API 文档与企业方案；围绕 Grok，官方公开能力已经延伸到文本、视觉、语音、图像与视频生成、搜索和工具调用，因此仓库中凡是讨论 Grok、grok.com、X 内嵌 AI 功能或 xAI 官方接口的文档，通常都与该词条直接相关。

## 实践要点

- 讨论 xAI 时，要区分公司层、产品层与平台层：`xAI` 是公司与模型生态，`Grok` 是核心产品族，`X` 是其重要分发与使用场景之一。
- 若文档只泛泛提到马斯克或 X 平台，不应机械回写到 `xAI`；只有在内容直接涉及 xAI 公司、Grok 产品或官方 API 生态时才适合关联。
- 在工程语境下，xAI 相关资料通常落在 llm、多模态、搜索、工具调用与 Agent 化交互这些能力边界上。
- 由于 xAI 产品线与命名迭代快，涉及模型能力、产品入口或企业特性时，应优先核对 `x.ai` 与 `docs.x.ai`，而不是沿用过时的二手概括。
