## 定义

Claude Code 是 Anthropic 推出的智能代理编程工具，面向代码库理解、文件编辑、命令执行与跨端协作场景，把 Claude 的模型能力产品化为可直接进入工程工作流的开发入口。

## 核心内涵

Claude Code 的关键不在“对话式写代码”，而在让模型直接参与真实开发闭环：读取仓库上下文、修改文件、运行命令、衔接 IDE 与终端，并接入 Slack、CI/CD、MCP 等现有工具链。

在仓库语境里，Claude Code 通常代表 Anthropic 在 agentic coding 方向上的终端优先产品形态。它强调多设备流转、会话迁移与可配置自动化，适合承接从探索代码库到提交修复、发起 PR 的多步任务。

## 实践要点

- 使用 Claude Code 时，应把它视为“工程执行入口”，而不是单纯聊天界面。
- 典型落地方式包括终端、IDE、桌面应用与 CI 场景，并常结合 MCP、自定义指令、技能与 hooks 扩展能力。
- 当它具备文件编辑和命令执行权限时，仍需结合代码审查、权限控制与 CI 验证来约束执行风险。
