# memory

## 文档信息

- 类型：主题词条
- 更新日期：2026-02-23

## 定义

memory 指 AI 系统在多轮交互与长期任务中，对历史信息进行存储、检索与更新的能力。
memo 在本仓库中视为 memory 的同义表达，不单独设元语词条。

## 核心内涵

在 Agent 与 LLM 应用中，memory 不是单一技术点，而是“状态持续化 + 语义检索 + 任务回放”的组合能力。它决定了系统能否跨会话保持上下文一致性、减少重复输入、支持长期规划与个性化协作。

## 实践要点

- 区分短期记忆与长期记忆，避免把全部上下文塞入单轮窗口。
- 结合结构化状态与语义检索，提升召回准确率。
- 对记忆写入与淘汰建立规则，控制噪音和陈旧信息。
- 在评测中单独验证记忆读写一致性与跨任务稳定性。
