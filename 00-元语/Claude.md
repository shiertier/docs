## 定义

Claude 是 Anthropic 的大语言模型家族与开发平台品牌，覆盖对话、编码、工具调用与智能体构建等场景。

## 核心内涵

官方文档将 Claude 定位为一组面向不同性能与成本层级的模型，并明确支持文本与图像输入、文本输出、多语言任务、长上下文与开发者接入能力。在仓库语境里，Claude 不只指聊天产品，也常指 Claude Code、Claude Agent SDK 以及围绕工具调用和多智能体编排形成的一整套工程实践。

Claude 的代表性优势通常体现在高信息密度文本理解、代码库导航、需求拆解、长上下文保持以及对安全与对齐约束的强调。它常被用于需要多轮状态维护、复杂文档分析和 agent 工作流协同的任务。

## 实践要点

- 使用 Claude 时，通常要把它理解为“模型家族 + 开发入口”，而不是单一聊天界面。
- 在工程场景中，Claude 往往通过 Claude Code、API、SDK、MCP 等形态进入真实工作流，适合承接长上下文分析、工具协作和多步任务拆解。
- Claude 与 Codex 经常出现在同一条开发链路里，但分工应基于任务类型、工具链和团队治理要求来决定，而不是机械按品牌切分。
