# Claude

## 文档信息

- 类型：元语词条
- 更新日期：2026-02-23

## 定义

Claude 是 Anthropic 的模型与 agent 生态品牌，在本仓库中常与 Codex 并行出现于多智能体协作工具与实践讨论。

## 核心内涵

Claude 系列模型以其长上下文窗口、细致的指令遵循能力和注重安全性的对齐策略而著称。在智能体工程中，Claude 常被用于需要深度逻辑推理、复杂文本分析、大规模代码库理解和多轮对话状态维护的场景，能够有效处理高信息密度的任务。

## 实践要点

- 常见定位是与 Codex 形成互补：不同模型策略、不同工具链偏好。
- 多 agent GUI 工具通常同时提供 Claude 与 Codex 通道。
- 在实际开发中，可以利用 Claude 处理需求分析与架构设计，而将具体的代码生成与终端执行交由其他专长模型，形成异构智能体协作。
