# cli

## 文档信息

- 类型：元语词条
- 更新日期：2026-02-22

## 定义

cli 指命令行界面（Command Line Interface），用于通过终端命令与程序交互。

## 核心内涵

CLI 以“命令 + 参数 + 输出”的文本交互为核心，强调可组合、可脚本化与可复现，适合自动化流程、远程环境与批处理任务。相比图形界面，CLI 的优势在于更容易串联工具链并纳入版本化工作流。

## 实践要点

使用 CLI 时应优先保证命令可复现与可回滚；涉及权限提升、系统修改或批量删除时，需要明确边界并进行二次确认。
