## 定义

v0 是 Vercel 推出的 AI 驱动开发平台与编程助手，通过对话界面把自然语言、截图、Figma 设计稿与现有代码上下文转成可运行、可部署的全栈 Web 应用；其生成与迭代能力对 Next.js、React、Tailwind CSS 与 shadcn/ui 尤其友好。

## 核心内涵

v0 的核心不是“把一句提示词变成一段代码”，而是把提示、视觉参考、现有项目、集成配置与部署路径串成一条从 prompt 到 production 的开发闭环。它既覆盖原型和界面生成，也覆盖后端逻辑、数据库或外部 API 接入、代码编辑与一键部署，因此更接近面向 Web 产品开发的对话式构建平台，而不只是静态页面生成器。它在交互方式上也带有明显的 agent 化特征：会围绕当前目标持续吸收上下文、生成结果并推动迭代。

## 辨析

在本仓库中，独立出现的 `v0` 默认指 Vercel 的产品名，而不是通用版本号。若上下文写成 `v0.1`、`v0.2`、`v0.3` 等，通常表示版本迭代，不应机械回链到本词条。
