# Codex

## 文档信息

- 类型：元语词条
- 更新日期：2026-02-23

## 定义

Codex 是 OpenAI 面向编码与任务执行场景的智能体能力与产品体系，强调工具调用、代码执行与工程闭环。

## 核心内涵

Codex 不仅仅是一个代码生成模型，更代表了一种以代码为核心媒介的智能体交互范式。它能够理解自然语言意图，将其转化为可执行的代码或终端命令，并在沙箱环境中运行，根据执行结果进行自我修正，从而完成端到端的软件工程任务。

## 实践要点

- 关注点不仅是“写代码”，还包括规划、执行、验证、修复与交付。
- 在多 agent 场景中，Codex 常作为 terminal-first 的工程执行体。
- 使用时需为其提供安全的执行环境，并结合严格的 CI 流程以控制潜在的执行风险，避免对生产环境造成意外影响。
